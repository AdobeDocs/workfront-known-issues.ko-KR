# 문서: 증명 워크플로우 아이콘이 표시되지 않습니다.

>[!NOTE]
>
>제품 팀이 현재 이 문제 해결 방법을 평가하고 있습니다. 이 문제가 해결되면 유지 관리 업데이트가 아닌 제품 공지 사항에 전달됩니다.

사용자가 새 [!DNL Workfront] 경험에서 증명 상태 아이콘 &quot;[!UICONTROL SOCD]&quot; [!UICONTROL 문서] 사용자가 페이지를 새로 고칠 때까지 나열합니다.
