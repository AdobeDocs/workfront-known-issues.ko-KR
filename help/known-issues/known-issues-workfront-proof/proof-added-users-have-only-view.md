# [!DNL Workfront Proof]: 증명에 추가된 사용자에게는 &quot;보기&quot; 권한만 있습니다

>[!NOTE]
>
>이 문제는 2022년 7월 25일에 수정되었습니다.

* [!UICONTROL 중재자]
* [!UICONTROL 검토자]
* [!UICONTROL 승인자]

_2022년 6월 1일에 처음 보고되었습니다._
