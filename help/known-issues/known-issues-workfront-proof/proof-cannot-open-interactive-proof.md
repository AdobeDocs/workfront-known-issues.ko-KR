# [!DNL Workfront Proof]: 대화형 증명을 열 수 없습니다.

사용자가 대화형 증명을 열려고 할 때 증명은 열리지 않고 사용자에게 다음 메시지가 표시됩니다.

&quot;[!UICONTROL 증명을 로드할 수 없습니다(501). 다시 시도하십시오]&quot;

이 문제는 사용자가 해당 버전의 [!DNL Workfront Proof].

_2022년 11월 3일에 처음 보고되었습니다._
