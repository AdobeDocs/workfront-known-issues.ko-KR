# Workfront Proof: 데스크탑 교정쇄 뷰어의 업데이트 섹션에 콘텐츠가 잘못 붙여넣어짐

>[!NOTE]
>
>이 문제는 2024년 10월 2일에 수정되었습니다.

데스크탑 교정쇄 뷰어에서 교정쇄의 업데이트 섹션에 콘텐츠를 붙여넣으려고 하면 콘텐츠가 클립보드에 복사된 내용과 일치하지 않습니다.

_2024년 8월 30일에 처음 보고되었습니다._
