# [!DNL Workfront Proof]: 요청자 및 검토자는 조직의 라이선스 수에 포함됩니다

사용자가 검토자 또는 요청자로 증명에 추가되면 &quot;[!UICONTROL 방문자]&quot; 권한 프로필. [!DNL Workfront Proof] 라이센스. 그러나 사용자가 추가되면 사용된 수의 [!DNL Workfront Proof] 라이센스가 증가합니다.
