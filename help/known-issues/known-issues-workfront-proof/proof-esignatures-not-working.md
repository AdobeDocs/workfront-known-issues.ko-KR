# Workfront 증명: 전자 서명이 작동하지 않음

>[!NOTE]
>
>이 문제는 2024년 4월 5일 토요일에 해결되었습니다.

<!--wf. wfp-->

사용자가 Workfront Proof에서 증명을 보고 e-signature를 사용하여 결정하려고 하면 e-signature가 유효하지 않으며 사용자가 결정에 서명할 수 없습니다.

_2024년 3월 26일 수요일에 처음 보고되었습니다._
