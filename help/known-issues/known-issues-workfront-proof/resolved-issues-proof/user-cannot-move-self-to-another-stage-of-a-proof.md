# 증명: 사용자는 증명의 다른 단계로 이동할 수 없습니다

>[!NOTE]
>
>이 문제는 2022년 6월 24일에 수정되었습니다.

사용자가 증명의 증명 워크플로우를 보고 자신을 증명의 다른 단계로 끌려고 하면 사용자의 이름이 원래 스테이지로 스냅되며 원하는 스테이지에 추가되지 않습니다.

2022년 6월 2일에 처음 보고되었습니다.
