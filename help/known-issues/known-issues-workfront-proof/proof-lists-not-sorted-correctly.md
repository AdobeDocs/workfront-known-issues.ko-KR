# [!DNL Workfront Proof]: 목록이 올바르게 정렬되지 않음

사용자가 [!DNL Workfront Proof]에서 정렬이 적용된 목록(예: [!UICONTROL 이름])을 볼 때 정렬이 정확하지 않습니다. 또는 정렬을 적용할 때 목록이 비어 있습니다.
