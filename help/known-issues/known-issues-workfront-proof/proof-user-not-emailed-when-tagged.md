# 증명: 증명 댓글에서 태그를 지정하면 사용자에게 이메일이 수신되지 않음

>[!NOTE]
>
>이 문제는 재현할 수 없어 종결되었습니다.

사용자가 증명 댓글에서 @이름을 사용하여 태그 지정되는 경우 이 사용자에게는 해당 댓글에 대한 이메일 알림이 수신되지 않습니다.

_2022년 12월 8일에 처음 보고되었습니다._
