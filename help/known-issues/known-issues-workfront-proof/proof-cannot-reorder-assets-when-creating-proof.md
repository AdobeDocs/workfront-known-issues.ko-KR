# [!UICONTROL Workfront 증명]: 증명을 만들 때 에셋을 재정렬할 수 없음

사용자가에서 [!UICONTROL Workfront 증명] 증명에 결합될 에셋의 순서를 재지정하려고 하면 파일을 드래그할 수 없고 파일 순서는 변경되지 않습니다.

증명에 대한 에셋 순서 조정은에서 예상대로 작동합니다. [!DNL Workfront].
