# [!DNL Workfront Proof]: 사용자가 증명을 태그 지정할 때 알림이 수신되지 않음

>[!NOTE]
>
>이 문제는 2022년 8월 16일에 수정되었습니다.

사용자가 증명 댓글을 태그 지정할 때 해당 댓글에 대한 이메일 알림이 수신되지 않습니다.

**2022년 4월 28일에 처음 보고되었습니다.**
