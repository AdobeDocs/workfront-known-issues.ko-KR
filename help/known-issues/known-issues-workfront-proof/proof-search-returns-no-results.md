# [!DNL Workfront Proof]: 검색 결과 없음

사용자가 [!DNL Workfront Proof]를 반환하면 검색 결과가 반환되지 않습니다. 검색을 통해 사용자가 선택할 수 있는 증명을 제공할 수 있지만 사용자가 검색을 마치면 이러한 제안이 결과에 표시되지 않습니다.

_2023년 4월 26일에 처음 보고되었습니다._
