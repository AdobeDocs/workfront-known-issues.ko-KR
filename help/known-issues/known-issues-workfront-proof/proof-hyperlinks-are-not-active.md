# [!DNL Workfront Proof]: 대화형 증명 내의 하이퍼링크는 활성 상태가 아닙니다

>[!NOTE]
>
>이 문서는 2022년 8월 18일에 수정되었습니다.

사용자가 대화형 증명을 보고 링크가 들어 있는 링크 또는 단추를 클릭해도 해당 링크나 단추가 연결되어 있는 페이지로 이동하지 않습니다.
