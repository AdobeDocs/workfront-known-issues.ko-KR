# [!DNL Workfront Fusion]: 시나리오 실행 기록에 전체 기록이 표시되지 않음

사용자가 시나리오 실행 기록을 볼 때 전체 실행이 기록에 포함되지 않습니다. 시나리오가 성공적으로 실행되었지만 실행 기록에는 실행 중에 실행된 모든 모듈에 대한 정보가 포함되지 않습니다.

_2023년 2월 26일에 처음 보고되었습니다._
