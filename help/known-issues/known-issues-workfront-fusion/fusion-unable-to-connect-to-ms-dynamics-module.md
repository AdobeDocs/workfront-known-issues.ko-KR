# Workfront Fusion: Fusion의 [!DNL MS Dynamics] 모듈에 연결할 수 없음

>[!NOTE]
>
> 이 문제는 2022년 8월 12일에 수정되었습니다.

[!DNL Fusion]의 [!DNL MS Dynamics] 모듈에 연결할 수 없어 새 연결과 기존 연결 모두에서 “연결 확인 실패” 오류가 표시됩니다.

_2022년 8월 10일에 처음 보고되었습니다._
