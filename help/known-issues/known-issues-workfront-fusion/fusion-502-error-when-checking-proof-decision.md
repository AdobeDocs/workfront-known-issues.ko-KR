# [!DNL Workfront Fusion]: 시나리오에 [!DNL Workfront]에서 증명 결정을 확인하는 모듈이 포함된 경우 502 오류 발생

>[!NOTE]
>
>이 문제는 2022년 8월 22일에 수정되었습니다.

[!DNL Workfront]에 증명 결정을 확인하는 모듈을 포함하도록 시나리오가 실행되면 해당 모듈은 502 오류를 반환합니다.

_2022년 7월 28일에 처음 보고되었습니다._
