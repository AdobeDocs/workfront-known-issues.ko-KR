# Workfront Fusion: 웹후크 업데이트가 자동 저장되지 않음

사용자가 Workfront > 보기 이벤트 모듈과 같이 웹후크를 변경할 때, 변경 사항이 자동 저장되지 않습니다.

사용자는 웹후크의 변경 사항을 보존하기 위해 **시나리오를 저장해야 합니다**.

**해결 방법**

시나리오를 저장합니다.

_2025년 7월 17일에 처음 보고되었습니다._
