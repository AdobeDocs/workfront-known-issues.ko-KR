# Workfront Fusion: 실행 세부 정보가 처리 중으로 표시됨

사용자가 실행 세부 정보를 보려고 할 때 실행이 완료된 후 오랜 시간이 지나야 “처리” 배지가 표시됩니다.

사용자가 시나리오 세부 정보를 보려고 하면 “로그 세부 정보가 존재하지 않습니다.”라는 메시지가 표시됩니다.

_2024년 8월 22일에 처음 보고되었습니다._
