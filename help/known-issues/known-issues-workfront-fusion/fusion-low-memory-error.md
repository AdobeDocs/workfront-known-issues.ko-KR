# Workfront Fusion: 낮은 메모리 오류 발생

>[!NOTE]
>
>이 문제는 해결되었습니다.

낮은 메모리 오류와 함께 시나리오 실행이 실패합니다. 이전에 시나리오가 정상적으로 실행된 경우에도 이러한 오류가 발생할 수 있습니다.

**해결 방법**

해당 시나리오를 여러 개의 작은 시나리오로 나눕니다.

_2023년 9월 15일에 처음 보고되었습니다._
