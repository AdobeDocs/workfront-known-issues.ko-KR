# 보고서: 그룹화 이름에 / 또는 \

사용자가 보고서에서 차트를 볼 때 차트에서 그룹화를 클릭할 때 해당 그룹화의 이름에 / 또는 \ 이 있으면 열려 있는 세부 정보에 그룹화의 항목이 표시되지 않고 &quot;표시할 데이터 없음&quot; 메시지가 표시됩니다.

_2024년 6월 17일 화요일에 처음 보고되었습니다._
