# 보고서: 그룹 이름에 / 또는 \가 있는 경우 &quot;표시할 데이터 없음&quot;

사용자가 보고서에서 차트를 보면서 차트의 그룹화를 클릭했을 때 해당 그룹의 이름에 / 또는 \가 있는 경우, 열리는 세부 정보에 그룹의 항목이 표시되지 않고 사용자에게는 &quot;표시할 데이터 없음&quot; 메시지가 표시됩니다.

_2024년 6월 17일에 처음 보고되었습니다._
