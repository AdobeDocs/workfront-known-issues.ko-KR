# 프로젝트: 내보낼 때 통화 기호가 올바르지 않음

>[!NOTE]
>
>이 문제는 2024년 8월 15일에 수정되었습니다.

사용자가 문제를 내보낼 때 내보낸 내용의 통화 기호가 프로젝트나 문제에 설정된 통화 기호와 일치하지 않습니다.

_2024년 7월 12일에 처음 보고되었습니다._
