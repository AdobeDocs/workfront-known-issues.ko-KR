# 할당: 팀 할당이 저장되지 않음

>[!NOTE]
>
>이 문제는 2025년 12월 15일에 해결되었습니다.

사용자가 Workfront에서 작업 항목에 팀을 할당할 때 팀 할당이 저장되지 않으며, 해당 팀에 작업에 대한 알림이 전송되지 않습니다.

_2025년 11월 13일에 처음 보고되었습니다._
