# 요청: 새 요청 경험의 초안에 문제가 있음

>[!NOTE]
>
>이 문제는 2026년 2월 9일에 수정되었습니다.

새 요청 경험의 초안과 관련하여 다음 문제가 보고되었습니다.

* 초안으로 만들어진 후 제출된 요청에는 요청 목록에 제목의 첫 글자만 표시됩니다.
* 삭제된 초안은 요청 목록에서 제거되지 않으므로 “초안 검색 실패” 오류가 발생합니다.

_2025년 12월 30일에 처음 보고되었습니다._
