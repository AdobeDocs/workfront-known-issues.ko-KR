# 증명: 미리보기 환경에서 증명을 생성할 수 없음

>[!NOTE]
>
>이 문제는 미리보기 샌드박스 환경에서만 보고되었습니다.

사용자가 미리보기 샌드박스 환경에서 증명을 만들려고 할 때 증명이 생성되지 않고 “생성 실패” 메시지가 표시됩니다.

_2024년 10월 17일에 처음 보고되었습니다._
