# 교정쇄: 미리보기 환경에서 교정쇄를 생성할 수 없음

>[!NOTE]
>
>이 문제는 재현할 수 없어 종결되었습니다.

사용자가 미리보기 샌드박스 환경에서 교정쇄를 만들려고 할 때 교정쇄가 생성되지 않고 “생성 실패” 메시지가 표시됩니다.

_2024년 10월 17일 금요일에 처음 보고되었습니다._
