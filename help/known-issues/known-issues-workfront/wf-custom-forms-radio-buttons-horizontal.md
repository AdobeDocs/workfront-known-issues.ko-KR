# 사용자 정의 양식: 라디오 버튼이 가로로 나열됨

>[!NOTE]
>
>이 문제는 2025년 1월 9일 금요일에 해결되었습니다.

사용자가 라디오 단추가 포함된 사용자 정의 양식을 볼 때 라디오 단추가 세로로 표시되지 않고 가로로 표시됩니다.

_2024년 12월 12일에 처음 보고되었습니다._
