# 홈: 위임된 작업은 공유되지 않고 홈 작업 목록에 표시되지 않음

사용자가 홈 작업 목록에서 다른 사용자에게 작업을 위임하면 해당 작업은 위임받은 사용자와 공유되지 않으며 위임받은 사용자의 홈 작업 목록에 표시되지 않습니다. 이는 위임받은 사용자가 위임된 작업에 대한 액세스 권한이 없는 경우에 발생합니다.

_2023년 6월 1일에 처음 보고되었습니다._
