# 문서: 문서 개수가 정확하지 않음

사용자가 문서가 첨부된 오브젝트를 볼 때 왼쪽 탐색의 문서 탭에 잘못된 수의 문서가 표시될 수 있습니다. 표시되는 숫자는 루트 폴더에 있는 문서의 수입니다. 하위 폴더에 있는 문서는 계산되지 않습니다.

_2024년 6월 20일에 처음 보고되었습니다._
