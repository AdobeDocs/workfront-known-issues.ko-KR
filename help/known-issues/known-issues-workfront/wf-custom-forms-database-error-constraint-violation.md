# 사용자 정의 양식: 사용자 정의 필드를 삭제할 수 없음

>[!NOTE]
>
>이 문제는 2023년 9월 28일에 수정되었습니다.

사용자가 사용자 정의 필드를 삭제하려고 하면 이를 삭제할 수 없고 “[!UICONTROL 제한 사항 위반으로 인한 데이터베이스 오류]”가 발생했음을 알리는 메시지가 표시됩니다.

_2023년 8월 14일에 처음 보고되었습니다._
