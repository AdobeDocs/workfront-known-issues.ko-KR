# 보드: 예상되는 카드가 보드에 표시되지 않음

>[!NOTE]
>
>이 문제는 2023년 11월 9일에 해결되었습니다.

사용자가 보드를 볼 때 표시되어야 하는 카드가 모두 표시되지 않습니다. 보드 페이지의 보드에 표시된 카드 수가 정확한 경우에도 해당 문제가 발생합니다.

_2023년 10월 10일에 처음 보고되었습니다._
