# 설정: 다른 사용자로 로그인하면 “다음으로 로그인됨” 배너가 나타나지 않습니다.

관리자가 다른 사용자로 로그인하면 “다음으로 로그인됨” 배너가 나타나지 않습니다. 사용자는 다른 사용자로 로그인되어 있다는 알림을 받지 못하며 해당 사용자의 계정에서 로그아웃할 수 없습니다.
