# 사용자 정의 양식: 필드 표시 논리를 올바르게 따르지 않음

>[!NOTE]
>
>이 문제는 2025년 10월 22일에 해결되었습니다.

사용자가 표시 논리를 포함하는 사용자 정의 양식을 채울 때(일부 필드는 다른 필드의 값에 따라 표시되거나 표시되지 않음), 해당 필드 논리를 따르는 방식으로 필드가 표시되지 않습니다.

_2025년 10월 6일에 처음 보고되었습니다._
