# 프로젝트, 작업 및 문제: 상속된 권한은 표시되지 않음

>[!NOTE]
>
>이 문제는 2024년 4월 22일 화요일에 해결되었습니다.

사용자가 오브젝트를 공유할 때 상속된 권한을 표시하는 카드가 없습니다. 이 문제는 다음 사용자에 대해 보고되었습니다.

* 오브젝트의 할당자
* 개체를 공유할 이름을 입력하는 사용자입니다.

_2024년 1월 9일 수요일에 처음 보고되었습니다._
