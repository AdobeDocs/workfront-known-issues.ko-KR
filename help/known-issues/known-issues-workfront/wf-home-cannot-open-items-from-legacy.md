# 홈: [!UICONTROL 홈 캘린더] 또는 [!UICONTROL 작업 목록]의 항목을 열 수 없습니다.

>[!NOTE]
>
>이 문제는 2024년 7월 18일 금요일에 해결되었습니다.

사용자가 [!UICONTROL 홈 캘린더] 또는 [!UICONTROL 홈 작업 목록]에서 작업 항목이나 증명을 열려고 하면 해당 항목이 열리지 않습니다.

**해결 방법**

페이지를 새로 고칩니다.
