# 통합: Workfront의 Slack 알림이 트리거되지 않음

Workfront에서 Slack 알림을 트리거해야 하는 이벤트가 발생할 때 알림이 전송되지 않습니다.

이 문제는 다음 이벤트에 대해 보고되었습니다.

* 사용자가 영향을 받는 사용자에게 할당된 작업 또는 프로젝트에 대해 댓글을 남깁니다.
* 영향을 받는 사용자가 Workfront의 댓글에 언급되거나 태그 지정됩니다.

사용자의 알림 환경 설정이 Slack에서 알림을 보내도록 구성된 경우에도 이 문제가 발생할 수 있습니다.

_2026년 2월 2일 화요일에 처음 보고되었습니다._
