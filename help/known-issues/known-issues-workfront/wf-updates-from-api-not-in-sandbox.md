# 업데이트: API를 통해 작성된 댓글은 샌드박스 환경에서 표시되지 않음

>[!NOTE]
>
>이 문제는 2024년 4월 29일 화요일에 해결되었습니다.

Workfront API를 통해 댓글이 작성되면 해당 댓글이 프로덕션 환경에 표시됩니다. 단, 샌드박스 환경에서는 댓글이 보이지 않습니다.

_2024년 4월 8일에 처음 보고되었습니다._
