# 업데이트: API를 통해 작성한 댓글은 샌드박스 환경에 표시되지 않습니다

Workfront API를 통해 댓글을 달면 해당 댓글이 프로덕션 환경에 표시됩니다. 하지만 주석은 샌드박스 환경에 표시되지 않습니다.

_2024년 4월 8일 화요일에 처음 보고되었습니다._
