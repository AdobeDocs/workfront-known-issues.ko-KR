# 문서: 문서를 다른 폴더로 이동할 수 없음

사용자가 문서를 객체 폴더로 이동하면 해당 객체를 다른 폴더로 이동할 수 없습니다.

**해결 방법:**

이동 옵션을 사용하여 문서를 상위 객체로 이동한 다음 올바른 폴더로 이동합니다.
