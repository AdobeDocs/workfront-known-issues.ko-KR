# 증명: 증명 댓글에 사용자를 태그 지정할 수 없음

사용자가 증명 댓글에서 다른 사용자를 태그 지정하려고 할 때 사용자 이름을 입력하려고 하면 목록에 사용자가 표시되지 않고 태그 지정을 선택할 수 없습니다.

**해결 방법**

사용자 이름의 첫 글자를 대문자로 입력해야 합니다. 이렇게 하면 사용자가 목록에 나타나고 사용자를 선택할 수 있습니다.
