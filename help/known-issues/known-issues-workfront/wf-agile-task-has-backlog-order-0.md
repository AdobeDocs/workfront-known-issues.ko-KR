# 애자일: 작업에 순서가 0인 Kanban 백로그 순서가 있음

>[!NOTE]
>
>이 문제는 2022년 12월 1일에 수정되었습니다.

사용자가 Kanban 팀의 백로그를 조회할 때 하나 이상의 작업에 백로그 순서가 0으로 표시됩니다.

_2022년 10월 7일에 처음 보고되었습니다._
