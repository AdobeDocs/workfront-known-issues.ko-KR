# 홈: [완료로 표시] 버튼 사용 시 내 문제 위젯에 대한 요약 창 닫힘

>[!NOTE]
>
>이 문제는 2024년 10월 10일에 수정되었습니다.

내 문제 위젯을 통해 열린 문제의 요약 창에서 [완료로 표시] 버튼을 사용하면 요약 창이 예기치 않게 닫힙니다.

_2024년 9월 25일에 처음 보고되었습니다._
