# 설정: 이메일 템플릿을 저장할 수 없음

>[!NOTE]
>
>이 문제는 2023년 2월 2일에 해결되었습니다.

사용자가 이메일 템플릿을 만들거나 편집하려고 하면 [!UICONTROL 저장] 버튼이 응답하지 않고 사용자는 해당 템플릿을 저장할 수 없습니다.

_2023년 1월 11일에 처음 보고되었습니다._
