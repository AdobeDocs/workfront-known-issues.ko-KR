# 업데이트: 새 댓글 달기 경험에 달린 댓글은 이전 경험에 표시되지 않습니다

사용자가 새 댓글 달기 경험에서 댓글을 달 때 해당 댓글이 새 경험의 댓글 영역에 표시되면 기존 댓글 달기 경험에 동일한 댓글이 표시되지 않을 수 있습니다. 이로 인해 기존 경험을 사용하는 사용자가 댓글을 놓칠 수 있습니다.

_2023년 9월 6일에 처음 보고되었습니다._
