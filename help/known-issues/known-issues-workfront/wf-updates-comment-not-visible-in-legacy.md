# 업데이트: 새로운 댓글 달기 경험에서 작성한 댓글이 기존 경험에 표시되지 않음

>[!NOTE]
>
>이 문제는 해결되었습니다.

사용자가 새로운 댓글 달기 경험에서 댓글을 작성하고 해당 댓글이 새로운 경험의 댓글 영역에 표시되면 동일한 댓글이 기존 댓글 달기 경험에 표시되지 않을 수 있습니다. 이로 인해 기존 경험을 사용하는 사용자는 댓글을 누락할 수 있습니다.

_2023년 9월 6일에 처음 보고되었습니다._
