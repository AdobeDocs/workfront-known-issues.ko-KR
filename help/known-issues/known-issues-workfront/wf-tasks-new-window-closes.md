# 작업: 새 작업을 만들 때 창이 닫힘

사용자가 + 새 작업 버튼을 사용하여 새 작업을 만들려고 하면 새 작업 창이 열리지만 사용자가 정보를 입력하기 전에 닫힙니다. 따라서 사용자는 이 메서드를 사용하여 새 작업을 만들 수 없습니다.
