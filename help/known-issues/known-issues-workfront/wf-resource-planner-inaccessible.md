# 리소스 플래너: 리소스 플래너에 액세스할 수 없음

>[!NOTE]
>
>이 문제는 2024년 11월 14일 금요일에 해결되었습니다.

사용자가 리소스 플래너에 액세스하려고 할 때 페이지를 로드하지 못하고 대신 오류 상자 또는 오류 메시지가 표시됩니다.

_2024년 11월 7일 금요일에 처음 보고되었습니다._
