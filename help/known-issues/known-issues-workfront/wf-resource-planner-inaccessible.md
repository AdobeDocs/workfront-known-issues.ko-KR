# 리소스 플래너: 리소스 플래너에 액세스할 수 없음

>[!NOTE]
>
>이 문제는 2024년 11월 14일에 해결되었습니다.

사용자가 리소스 플래너에 액세스하려고 하면 페이지가 로드되지 않고 대신 오류 상자나 오류 메시지가 표시됩니다.

_2024년 11월 7일에 처음 보고되었습니다._
