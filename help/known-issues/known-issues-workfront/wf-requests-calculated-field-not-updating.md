# 요청: 요청 작성 시 계산된 필드가 업데이트되지 않음

>[!NOTE]
>
>이 문제는 2024년 5월 20일에 해결되었습니다.

사용자가 요청과 관련된 사용자 정의 양식을 작성할 때 양식에 계산된 필드가 포함되어 있으면 해당 필드가 업데이트되지 않습니다. 사용자는 양식이 제출될 때까지 계산된 값을 볼 수 없습니다.

_2024년 3월 18일에 처음 보고되었습니다._
