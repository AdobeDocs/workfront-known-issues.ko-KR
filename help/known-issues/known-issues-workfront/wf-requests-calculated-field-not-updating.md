# 요청: 요청을 작성할 때 계산된 필드가 업데이트되지 않음

사용자가 요청과 관련된 사용자 정의 양식을 작성할 때 양식에 계산된 필드가 포함된 경우 필드가 업데이트되지 않습니다. 양식을 제출한 후 계산된 값을 볼 수 없습니다.

_2024년 3월 18일 화요일에 처음 보고되었습니다._
