# 증명: 증명의 새 버전을 만들 수 없음

>[!NOTE]
>
>이 문제는 2023년 10월 26일에 수정되었습니다.

사용자가 증명의 새 버전을 만들려고 하면 새 버전이 만들어지지 않고 사용자에게 다음과 같은 오류 메시지가 표시됩니다.

“[!UICONTROL 증명 생성 도중 오류 발생]”

**해결 방법**

새 증명 버전 대신 새 증명을 만듭니다.

_2023년 10월 6일에 처음 보고되었습니다._
