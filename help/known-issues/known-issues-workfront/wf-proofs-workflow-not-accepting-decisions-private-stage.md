# 증명: 비공개 워크플로 단계에서 결정을 만들 수 없음

>[!NOTE]
>
>이 문제는 2024년 5월 9일 금요일에 해결되었습니다.

자동화된 워크플로 단계가 비공개인 경우 사용자가 문서 세부 정보 > 증명 워크플로 단계에서 증명 결정을 만들 수 없습니다.

증명 뷰어에서의 증명 결정은 예상대로 작동합니다.

**해결 방법**

증명 뷰어를 사용하여 결정을 만드십시오.

_2024년 4월 23일에 처음 보고되었습니다._
