# 교정쇄: 완료된 교정쇄가 [!UICONTROL 승인 대기 중] 위젯에 남아 있음

>[!NOTE]
>
>이 문제는 재현할 수 없어 종결되었습니다.

사용자가 교정쇄에 대한 결정을 내리면 해당 교정쇄는 새로운 홈 환경의 사용자 [!UICONTROL 홈] 페이지에서 [!UICONTROL 승인 대기 중] 위젯에 남아 있습니다.

_2023년 9월 21일 금요일에 처음 보고되었습니다._
