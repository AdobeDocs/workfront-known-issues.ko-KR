# 사용자 정의 양식: 요청 양식에 표시 논리가 올바르게 표시되지 않음

>[!NOTE]
>
>이 문제는 2026년 2월 13일 토요일에 해결되었습니다.

사용자가 요청에 연결된 사용자 정의 양식을 채울 때 표시 논리가 따르지 않고 일부 필드가 예상대로 표시되지 않을 수 있습니다.

**해결 방법**

요청 초안을 닫은 다음 다시 엽니다.

_2026년 1월 15일 금요일에 처음 보고되었습니다._
