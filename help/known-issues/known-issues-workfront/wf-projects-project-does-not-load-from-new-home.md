# 프로젝트: 새 [!UICONTROL 홈] 환경에서 프로젝트를 로드하면 프로젝트가 비어 있음

>[!NOTE]
>
>이 문제는 2023년 9월 7일에 수정되었습니다.

사용자가 새 홈 환경의 [!UICONTROL 홈] 페이지에서 프로젝트를 클릭하면 프로젝트 페이지가 로드되지 않습니다.

이는 사용자가 다른 사용자로 로그인한 후 해당 다른 사용자로 로그아웃하고 자신의 [!UICONTROL 홈] 페이지로 돌아왔을 때 발생합니다.

**해결 방법**

페이지를 다시 로드합니다.

_2023년 8월 29일에 처음 보고되었습니다._
