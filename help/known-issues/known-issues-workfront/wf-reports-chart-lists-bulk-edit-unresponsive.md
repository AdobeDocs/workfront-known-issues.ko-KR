# 보고서: 차트 목록에서 일괄 편집이 응답하지 않음

>[!NOTE]
>
>이 문제는 2025년 2월 13일에 해결되었습니다.

사용자가 보고서에서 차트를 보고 있을 때 목록에서 여러 항목을 선택하여 대량 편집하면 커서가 사라지고 대량 편집을 위한 텍스트 상자가 응답하지 않게 됩니다.

_2024년 10월 24일에 처음 보고되었습니다._
