# 증명: 단계 활성화 문제

[!UICONTROL 증명 워크플로]의 새 단계가 활성화되면 해당 단계가 증명 워크플로 영역에 활성 상태로 표시됩니다. 그러나 새 단계에서는 문서가 사용자와 공유되지 않으며, [!UICONTROL 증명 활동] 영역에는 단계가 활성화되었거나 문서가 공유되었다는 내용이 표시되지 않습니다.
