# 타임시트: 댓글이 저장되지 않음

>[!NOTE]
>
>이 문제는 2022년 8월 2일에 수정되었습니다.

사용자가 타임시트의 시간 항목에 댓글을 추가하면 댓글이 저장되지 않습니다.

_2022년 7월 29일에 처음 보고되었습니다._
