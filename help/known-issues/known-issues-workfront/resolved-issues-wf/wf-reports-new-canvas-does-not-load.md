# 보고서: [!UICONTROL 새 보고 캔버스]가 로드되지 않음

>[!NOTE]
>
>이 문제는 2022년 7월 25일에 해결되었습니다.

사용자가 [!UICONTROL 새 보고 캔버스]를 로드하려고 하면 “[!UICONTROL 데이터를 로드하는 중]” 화면이 표시되지만 [!UICONTROL 새 보고 캔버스]가 로드되지 않습니다.

_2022년 7월 4일에 처음 보고되었습니다._
