# [!UICONTROL 설정]: 새 상태를 만들 수 없음

>[!NOTE]
>
>이 문제는 2022년 8월 1일에 수정되었습니다.

사용자가 새 상태를 만들려고 할 때 [!UICONTROL 설정]의 상태 목록에 상태가 표시되지 않습니다. 이 문제는 새 상태와 삭제된 상태의 키가 동일할 때 발생합니다.

_2022년 7월 12일에 처음 보고되었습니다._
