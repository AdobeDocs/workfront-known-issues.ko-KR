# 업데이트: 사용자에게 태깅한 다음 멀리 탐색하면 태그가 지정된 사용자가 제거됩니다

>[!NOTE]
>
>이 문제는 2022년 8월 4일에 수정되었습니다.

사용자가 다른 사용자에게 입력하여 업데이트 스트림에 알리고 다른 탭으로 이동한 다음 업데이트 스트림으로 돌아가면 태그가 지정된 사용자가 제거됩니다.

_2022년 7월 7일에 처음 보고되었습니다._
