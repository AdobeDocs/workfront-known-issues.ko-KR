# 템플릿: 템플릿을 첨부할 때 작업을 선택 취소할 수 없습니다

>[!NOTE]
>
>이 문제는 2022년 7월 28일에 수정되었습니다.

사용자가 템플릿을 연결하고 사용자 지정할 때 포함하지 않으려는 작업의 선택을 취소하라는 메시지가 표시됩니다. 그러나 선택된 것으로 표시되지 않고 사용자가 선택 취소할 수 없습니다.

_2022년 6월 23일에 처음 보고되었습니다._
