# 작업: 작업 기간을 인라인 편집할 때 경과 시간을 선택할 수 없음

>[!NOTE]
>
>이 문제는 2022년 7월 21일에 수정되었습니다.

사용자가 작업 목록을 보면서 [!UICONTROL 작업 기간]을 편집하려고 할 때 다음 기간 단위를 사용할 수 없습니다.

* [!UICONTROL 경과 시간 (분)]
* [!UICONTROL 경과 시간]
* [!UICONTROL 경과 일수]
* [!UICONTROL 경과 주 수]
* [!UICONTROL 경과 기간 (월)]

_2022년 7월 1일에 처음 보고되었습니다._
