# 요청: 템플릿을 사용하여 프로젝트로 전환되는 요청이 템플릿의 그룹이 아니라 요청 대기열의 그룹을 표시함

사용자가 템플릿을 사용하여 요청을 프로젝트로 전환할 때 새로 만들어지는 프로젝트가 템플릿에 할당된 그룹이 아니라 요청 대기열을 소유한 그룹에 연결됩니다. 이는 프로젝트를 만들 때 템플릿에 연결된 그룹이 [!UICONTROL 그룹] 필드에 채워지는 경우에도 발생합니다.

_2022년 7월 15일에 처음 보고되었습니다._
