# 요청: 템플릿이 있는 프로젝트로 변환된 요청은 템플릿의 그룹이 아니라 요청 큐의 그룹을 표시합니다

사용자가 템플릿을 사용하여 요청을 프로젝트로 변환하면 새로 생성된 프로젝트는 템플릿에 할당된 그룹이 아니라 요청 큐를 소유하는 그룹과 연결됩니다. 이 문제는 프로젝트를 만들 때 템플릿과 연결된 그룹이 [!UICONTROL 그룹] 필드.
