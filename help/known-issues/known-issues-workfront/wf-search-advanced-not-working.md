# 검색: 고급 검색이 예상 결과를 반환하지 않음

사용자가 고급 검색을 사용할 때 예상 결과가 반환되지 않습니다. 이 문제는 고급 검색에 적용된 필터가 없는 경우에도 발생할 수 있습니다.

**해결 방법**

기본 검색은 예상대로 작동합니다.

_2025년 5월 27일 수요일에 처음 보고되었습니다._
