# 작업: 전임 작업 정보에 링크가 포함되어 있지 않음

>[!NOTE]
>
>이 문제는 2024년 4월 23일 수요일에 해결되었습니다.

사용자가 자세한 정보를 보기 위해 전임 작업 위로 마우스를 가져다 대면 해당 정보에는 전임 작업에 대한 링크가 포함되지 않습니다.

_2024년 3월 22일에 처음 보고되었습니다._
