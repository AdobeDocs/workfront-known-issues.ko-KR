# 설정: “다음으로 로그인” 기능을 사용하면 빈 화면이 표시됨

>[!NOTE]
>
>이 문제는 2024년 8월 8일에 수정되었습니다.

관리자가 다른 사용자로 로그인하면 해당 사용자의 계정이 아닌 빈 화면이 표시됩니다.

**해결 방법**

화면을 새로 고칩니다. 여러 번 시도해야 할 수 있습니다.

_2024년 7월 17일에 처음 보고되었습니다._
