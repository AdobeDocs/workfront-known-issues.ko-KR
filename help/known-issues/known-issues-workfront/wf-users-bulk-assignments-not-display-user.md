# 사용자: 일괄 할당할 때 사용자 누락

사용자가 일괄 할당을 하려고 할 때 작업에 작업 역할이 할당되면 해당 작업 역할을 가진 사용자가 목록에 표시되지 않고 할당 사용자가 이를 선택할 수 없습니다.

이 동작은 업무 균형자에서도 보고되었습니다.

_2024년 5월 17일 토요일에 처음 보고되었습니다._
