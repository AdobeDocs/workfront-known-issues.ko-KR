# 목록: 목록에 모든 항목이 표시되지 않음

사용자가 오브젝트 목록을 보고 있는 동안 오른쪽 하단에서 모두를 선택하면(목록에 페이지가 지정되지 않음) 항목은 페이지가 지정된 상태로 유지되며 모든 오브젝트가 페이지에 표시되지 않습니다.

_2024년 4월 11일에 처음 보고되었습니다._
