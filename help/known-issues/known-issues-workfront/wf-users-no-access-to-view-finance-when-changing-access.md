# 사용자: 사용자 액세스를 업데이트할 수 없음

>[!NOTE]
>
>이 문제는 설계대로 작동하는 것이므로 종결되었습니다.

사용자가 프로젝트에서 다른 사용자의 액세스를 보기에서 관리로 업데이트하려고 할 때 액세스를 업데이트할 수 없고 다음과 같은 메시지가 표시됩니다.

&quot;액세스 권한이 부족하여 이 프로젝트에 대한 재무를 볼 수 없습니다.&quot;

_2025년 12월 11일에 처음 보고되었습니다._
