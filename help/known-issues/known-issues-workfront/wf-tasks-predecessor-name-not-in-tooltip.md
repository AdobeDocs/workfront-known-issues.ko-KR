# 작업: 전임 작업 도구 설명에 작업 이름이 표시되지 않음

>[!NOTE]
>
>이 문제는 2024년 3월 28일 금요일에 해결되었습니다.

사용자가 작업 목록을 볼 때 자세한 정보를 보기 위해 전임 작업 아이콘 위로 마우스를 가져가면 표시되는 도구 설명에 전임 작업의 이름이 표시되지 않습니다.

_2024년 3월 5일 수요일에 처음 보고되었습니다._
