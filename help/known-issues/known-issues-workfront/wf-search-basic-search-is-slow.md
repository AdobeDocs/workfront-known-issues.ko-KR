# 검색: 검색은 매우 느립니다

사용자가 기본 검색(검색 아이콘)을 사용하여 검색하면 검색 결과가 매우 느리게 반환됩니다.

**해결 방법**

고급 검색을 사용합니다. 고급 검색 링크는 검색 아이콘을 클릭할 때 표시되는 드롭다운 하단에 있습니다.

_2022년 11월 1일에 처음 보고되었습니다._
