# 검색: [!UICONTROL 검색] 매우 느림

사용자가 기본 검색을 사용하여 검색할 때( [!UICONTROL 검색] 아이콘) 검색 결과가 매우 느리게 반환됩니다.

**해결 방법**

[!UICONTROL 고급 검색]을 사용합니다. 링크 대상 [!UICONTROL 고급 검색] 은 을 클릭하면 표시되는 드롭다운 하단에 있습니다. [!UICONTROL 검색] 아이콘.
