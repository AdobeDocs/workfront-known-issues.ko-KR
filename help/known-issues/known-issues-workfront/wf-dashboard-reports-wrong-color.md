# 대시보드: 대시보드에 표시된 보고서 차트는 보고서에서 볼 때와 다른 색상을 사용합니다

사용자가 대시보드에서 보고서 차트를 볼 때 사용되는 색상은 사용자가 보고서 자체를 볼 때와 다릅니다. 이로 인해 사용자가 특정 색상에 의미가 있을 경우 혼동될 수 있습니다.
