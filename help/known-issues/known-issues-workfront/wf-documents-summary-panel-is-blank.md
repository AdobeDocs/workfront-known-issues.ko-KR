# 문서: 요약 패널이 비어 있습니다.

사용자 중 [문서] 목록을 보고 문서를 클릭하면 [요약] 패널이 열리지만 비어 있습니다. 사용자 요약 패널을 닫을 수 없습니다.

사용자 이 페이지 다시 로드하면 [요약] 패널이 예상대로 채워질 수 있습니다. 그러나 사용자 사용자가 다른 문서를 클릭하면 해당 문서의 요약 패널이 비어 있습니다.

**해결 방법**

[요약] 패널을 임시로 보려면 페이지 새로고침을 시도합니다.

_2025년 1월 13일 화요일에 처음 보고되었습니다._
