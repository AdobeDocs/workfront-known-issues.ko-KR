# 문서: 요약 패널이 비어 있음

>[!NOTE]
>
>이 문제는 2026년 2월 5일 금요일에 해결되었습니다.

사용자가 문서 목록을 볼 때 문서를 클릭하면 요약 패널이 열리지만 비어 있습니다. 요약 패널을 닫을 수 없습니다.

페이지를 다시 로드하면 요약 패널이 예상대로 채워질 수 있습니다. 그러나 사용자가 다른 문서를 클릭하면 해당 문서의 요약 패널이 비어 있습니다.

**해결 방법**

일시적으로 요약 패널을 보려면 페이지를 다시 로드해 보십시오.

_2025년 1월 13일 화요일에 처음 보고되었습니다._
