# [!UICONTROL 홈]: [!UICONTROL 프로젝트 이름] 는 [!UICONTROL 작업 목록]

>[!NOTE]
>
>제품 팀이 현재 이 문제 해결 방법을 평가하고 있습니다. 이 문제가 해결되면 유지 관리 업데이트가 아닌 제품 공지 사항에 전달됩니다.

사용자에게 증명에 대한 액세스 권한이 없으며 증명을 승인하라는 메시지가 표시되면 사용자에게 표시됩니다 [!UICONTROL 작업 목록] in [!UICONTROL 홈] 정렬 기준 [!UICONTROL 프로젝트 이름]를 입력하면 프로젝트 이름을 볼 수 없으며 대신 GUID가 표시됩니다.

_2021년 1월 5일에 처음 보고되었습니다._
