# 문서: 문서 업데이트에 증명 댓글이 표시되지 않음

>[!NOTE]
>
>이 문제는 설계대로 작동하는 것이므로 종결되었습니다.

사용자가 증명에 대한 댓글을 달면 해당 댓글이 Workfront의 증명 문서 업데이트 영역에 복사되지 않습니다.

_2024년 7월 23일에 처음 보고되었습니다._
