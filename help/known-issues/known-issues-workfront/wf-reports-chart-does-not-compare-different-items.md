# 보고서: 조합 차트에 동일한 데이터가 두 번 표시됩니다.

>[!NOTE]
>
>이 문제는 2022년 12월 15일에 해결되었습니다.

사용자가 조합 차트를 볼 때 차트가 두 값 집합을 비교하는 대신 하나의 값 집합을 두 번 표시합니다. 보고서 세부 정보에서 올바른 값 집합을 찾을 수 있습니다.

_2022년 11월 4일에 처음 보고되었습니다._
