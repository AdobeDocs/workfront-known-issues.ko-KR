# 로그인: 로그인 페이지가 로그아웃 랜딩 페이지로 리디렉션됨

>[!NOTE]
>
>이 문제는 2023년 11월 30일에 해결되었습니다.

사용자가 [!DNL Workfront]에 로그인을 시도할 때, 로그인 페이지 대신 로그아웃할 경우 방문하게 되는 페이지로 이동됩니다.

**해결 방법**

귀하의 [!DNL Workfront] URL을 입력하고 다시 로그인을 시도합니다.

_2023년 5월 10일에 처음 보고되었습니다._
