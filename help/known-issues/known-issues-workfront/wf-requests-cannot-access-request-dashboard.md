# 요청: 요청 페이지(&quot;../submitted&quot; URL)의 대시보드에 액세스할 수 없음

요청 페이지에서 대시보드에 액세스하려고 하면 URL에 대시보드 URL이 일시적으로 표시된 다음 제출된 URL(/submitted)로 빠르게 전환됩니다.
