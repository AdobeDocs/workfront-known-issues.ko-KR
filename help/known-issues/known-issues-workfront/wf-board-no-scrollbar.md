# 보드: 가로 스크롤 막대가 없음

사용자가 보드를 볼 때 가로로 스크롤하려고 하면 스크롤바가 없고 가로로 스크롤할 수 없습니다.

**해결 방법**

보드를 클릭하고 오른쪽 및 왼쪽 화살표를 사용하여 스크롤합니다
