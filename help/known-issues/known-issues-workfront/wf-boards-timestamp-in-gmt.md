# 보드: 카드에 업로드된 문서에 잘못된 타임스탬프가 표시됨

>[!NOTE]
>
>이 문제는 2024년 2월 16일 토요일에 해결되었습니다.

사용자가 카드에 문서를 업로드하면 타임스탬프가 사용자의 시간대가 아닌 GMT(UTC)입니다. “2시간 전”과 같은 상대적인 시간은 정확하지만 카드에 표시되는 실제 시간이 부정확합니다.

_2024년 1월 30일에 처음 보고되었습니다._
