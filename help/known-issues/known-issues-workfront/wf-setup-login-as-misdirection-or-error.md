# 설정: 다른 사용자로 로그아웃할 때 문제 발생

관리자가 다른 사용자로 로그인하고 로그아웃하려고 하면 다음 문제가 발생할 수 있습니다.

* 사용자가 계정 대신 로그인 페이지로 리디렉션됩니다
* 사용자에게 &quot;[!UICONTROL Workfront에 대한 액세스 권한이 없습니다.]&quot; 오류가 표시됩니다.

_2024년 7월 30일 수요일에 처음 보고되었습니다._
