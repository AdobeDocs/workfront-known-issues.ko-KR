# 증명: 증명 생성 시 시간이 오래 걸림

>[!NOTE]
>
>이 문제는 2023년 9월 21일에 수정되었습니다.

사용자가 증명을 생성하려고 하면 증명을 생성하는 데 시간이 오래 걸립니다. 증명 생성에 며칠이 걸릴 수 있습니다.

_2023년 8월 2일에 처음 보고되었습니다._
