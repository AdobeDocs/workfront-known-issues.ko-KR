# Workfront Mobile: SSO를 사용하지 않고 로그인할 수 없음

사용자가 Workfront Mobile 앱에 로그인할 때 SSO를 사용하여 로그인해야 합니다. 사용자는 Workfront 자격 증명 또는 Adobe ID을 사용할 수 없습니다.

이 문제는 앱의 iOS 버전을 사용할 때 보고되었습니다.

_2024년 7월 29일에 처음 보고되었습니다._
