# 타임시트: 개인 작업이 타임시트에 표시됨

사용자가 다음에 대한 작업을 만들 때 [!UICONTROL 할 일] 새로운 위젯 [!UICONTROL 홈] 경험, 해당 작업이 사용자의 타임시트에 표시됩니다. 이 문제는 작업에 기록된 시간이 없고 개인 프로젝트가 고정되지 않은 경우에도 발생합니다.

_2023년 10월 3일에 처음 보고되었습니다._
