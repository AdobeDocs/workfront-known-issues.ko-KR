# 프로젝트: 프로젝트의 날짜를 변경할 때 “[!UICONTROL 타임라인 계획 수립 모드]” 오류 발생

>[!NOTE]
>
>이 문제는 2022년 8월 25일에 수정되었습니다.

사용자가 [!UICONTROL 플랜 모드]가 [!UICONTROL 수동 저장] > [!UICONTROL 타임라인 계획 수립]으로 설정된 프로젝트에서 날짜를 변경하려고 하면 날짜가 변경되지 않고 오류가 표시됩니다.

“[!UICONTROL 타임라인 계획 수립 모드는 timelineDate를 로드하는 경우에만 사용할 수 있습니다. 무엇이 잘못되었는지 파악하여 수정할 수 있도록 [!DNL Workfront]에 문의해 주시기 바랍니다.]”

_2022년 8월 3일에 처음 보고되었습니다._
