# 업데이트: 회신을 제출하면 화면이 일시적으로 깜박임

사용자가 업데이트에 회신하고 회신을 제출하면 화면이 일시적으로 깜박입니다. 화면은 7초 동안 깜박이며, 그동안 사용자는 회신을 실행 취소할 수 있습니다. 7초 후에는 실행 취소 옵션을 더 이상 사용할 수 없으며 화면이 예상대로 표시됩니다.

_2023년 1월 5일에 처음 보고되었습니다._
