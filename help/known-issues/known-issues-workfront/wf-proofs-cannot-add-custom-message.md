# 증명: 사용자 정의 메시지를 추가하면 증명 뷰어에 오류가 발생합니다.

<!--This is on both the WF and WFP TOCs-->

>[!NOTE]
>
>이 문제는 2022년 12월 1일에 해결되었습니다.

사용자가 증명을 공유하고 사용자 정의 메시지를 추가하려고 하면 다음 오류가 발생합니다.

* 증명 뷰어가 증명을 확대합니다.
* 왼쪽 탐색 영역이 더 이상 반응하지 않습니다.

_2022년 11월 10일에 처음 보고되었습니다._
