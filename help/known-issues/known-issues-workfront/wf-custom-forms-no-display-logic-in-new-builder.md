# 사용자 정의 양식: 새 양식 디자이너에서 사용자 정의 양식을 편집할 때 표시 논리가 손실됨(Beta)

현재 새 양식 디자이너(베타)에서는 표시 논리를 사용할 수 없습니다. 따라서 새 양식 디자이너(베타)에서 기존 양식을 편집하면 양식의 기존 표시 논리가 손실됩니다.

_2023년 5월 23일에 처음 보고되었습니다._
