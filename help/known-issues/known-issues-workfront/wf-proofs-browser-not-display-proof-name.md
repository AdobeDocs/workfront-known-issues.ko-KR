# 증명: 증명 이름이 브라우저 탭에 표시되지 않음

사용자가 증명을 볼 때 증명 이름이 브라우저 탭에 표시되지 않습니다. 대신 브라우저 탭에 &quot;Workfront&quot;가 표시됩니다.

_2024년 12월 9일 화요일에 처음 보고되었습니다._
