# 검색: 참조 번호로 프로젝트를 검색할 수 없습니다.

>[!NOTE]
>
>이 문제는 2022년 7월 1일에 수정되었습니다.

사용자가 참조 번호를 사용하여 기존 프로젝트를 검색하려고 하면 검색이 프로젝트를 반환하지 않습니다.

_2022년 6월 27일에 처음 보고되었습니다._
