# 작업: 기록된 시간이 이동되면 이전 프로젝트에 남아 있습니다.

작업이 새 프로젝트로 이동되면 원래 프로젝트의 프로젝트 시간 합계에 이동된 작업의 시간이 포함됩니다. 작업은 여전히 새 프로젝트에서 기록된 시간을 제대로 유지합니다.
