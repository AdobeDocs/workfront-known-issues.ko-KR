# 홈: 계획된 시간이 분 단위로 표시됨

>[!NOTE]
>
>이 문제는 2024년 5월 23일에 해결되었습니다.

사용자가 새로운 홈 경험의 위젯에 계획된 시간 열을 추가하면 계획된 시간이 분 단위로 표시됩니다.

_2024년 5월 1일에 처음 보고되었습니다._
