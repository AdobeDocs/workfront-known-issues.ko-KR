# 리소스 계획: [!UICONTROL 자세히 보기] 버튼이 표시되지 않음

사용자가 리소스 플래너를 보고 [!UICONTROL 자세히 보기] 버튼을 클릭하고자 할 때 버튼이 표시되지 않습니다.

**해결 방법**

[!UICONTROL 자세히 보기] 버튼이 있을 것으로 예상되는 영역을 클릭합니다. 버튼은 표시되지 않지만 클릭할 수 있습니다.
