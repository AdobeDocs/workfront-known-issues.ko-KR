# 사용자 정의 양식: 요청을 프로젝트로 전환할 때 섹션 구분에 의해 숨겨진 필드의 데이터가 유지되지 않음

>[!NOTE]
>
> 이 문제는 2022년 8월 11일에 수정되었습니다.

사용자가 요청을 프로젝트로 전환할 때 표시 논리를 사용하여 숨길 수 있는 섹션 구분 내에 데이터가 있는 사용자 정의 양식이 요청에 포함되어 있는 경우 해당 섹션 내 데이터는 새 프로젝트에 이전되지 않습니다.

_2022년 7월 27일에 처음 보고되었습니다._
