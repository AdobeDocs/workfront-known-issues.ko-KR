# 증명: 증명 뷰어를 닫을 수 없음


<!--
>[!NOTE]
>
>This issue was fixed on July 18, 2024.
-->

사용자가 증명 뷰어에서 증명을 보고 오른쪽 상단의 X를 클릭하여 증명을 닫으려고 할 때 증명이 닫히지 않습니다.

**해결 방법**

다음 중 하나를 수행하십시오.

* 이메일에서 문서 이름을 클릭하여 해당 문서로 이동한 다음 증명을 엽니다. 이렇게 하면 X가 예상대로 작동할 수 있습니다.
* 증명을 다시 닫아 보십시오.
* 브라우저 탭을 닫아 증명을 닫습니다.

_2024년 5월 31일에 처음 보고되었습니다._
