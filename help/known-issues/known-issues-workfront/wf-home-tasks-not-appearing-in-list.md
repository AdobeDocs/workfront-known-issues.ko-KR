# [!UICONTROL 홈]: 항목이에 표시되지 않음 [!UICONTROL 홈 작업 목록]

사용자가 보기 [!UICONTROL 홈 작업 목록], 다음에 표시되어야 하는 항목 [!UICONTROL 작업 목록] 이(가) 없습니다. 예를 들어 사용자가 프로젝트로 이동하여 프로젝트에서 작업이 할당되었는지 확인할 수 있지만 이러한 작업은 사용자의 [!UICONTROL 홈 작업 목록].
