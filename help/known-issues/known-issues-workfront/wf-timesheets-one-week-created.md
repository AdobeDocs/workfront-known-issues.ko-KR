# 타임시트: 타임시트가 올바르게 생성되지 않음

사용자가 타임시트를 생성하려고 할 때 생성이 &quot;격주&quot;로 설정되면 1주만 생성됩니다.

**해결 방법**

설정 > 시스템 > 진단 > 타임시트 생성을 통해 타임시트를 수동으로 생성합니다.

_2025년 6월 2일 화요일에 처음 보고되었습니다._
