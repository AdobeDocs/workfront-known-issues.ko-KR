# 홈: 프로젝트 이름으로 “액세스 권한 없음”이 표시됨

사용자가 홈에서 “내 작업” 위젯을 볼 때 프로젝트 이름 열에 “액세스 권한 없음”이 표시됩니다. 해당 프로젝트의 이름이 “내 작업” 위젯에 표시되는 경우에도 이 문제가 발생합니다.
