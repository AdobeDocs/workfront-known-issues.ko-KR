# 홈: &quot;액세스 권한 없음&quot;이 프로젝트 이름으로 표시됨

사용자가 홈에서 내 작업 위젯을 볼 때 프로젝트 이름 열에 &quot;액세스 권한 없음&quot;이 표시됩니다. 이 문제는 프로젝트 이름이 &quot;내 작업&quot; 위젯에 표시되는 경우에도 발생합니다.
