# 업데이트: 댓글에 있는 링크가 빈 화면으로 이동합니다.

사용자가 댓글에 있는 링크를 클릭하면 해당 링크가 사용자를 빈 화면으로 보냅니다.

**해결 방법**

링크를 복사하여 별도의 페이지의 URL 표시줄에 붙여넣습니다.
