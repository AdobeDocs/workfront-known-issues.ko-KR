# 업데이트: 댓글의 링크가 빈 화면으로 이동함

<!--

>[!NOTE]
>
>This issue was fixed on April 25, 2024.

-->

사용자가 댓글에 있는 링크를 클릭하면 해당 링크가 사용자를 빈 화면으로 연결합니다.

**해결 방법**

링크를 복사하여 별도 페이지의 URL 표시줄에 붙여넣으십시오.

_2024년 4월 18일 금요일에 처음 보고되었습니다._
