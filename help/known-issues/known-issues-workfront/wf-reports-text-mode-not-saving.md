# 보고서: 텍스트 모드에서 변경한 내용이 보고서 또는 보기에 적용되지 않음

>[!NOTE]
>
>이 문제는 3월 28일에 수정되었습니다. 2024.

사용자가 텍스트 모드에서 보고서나 보기를 변경할 때 변경 사항이 저장되지 않고 보고서나 보기에 적용됩니다.

_2024년 2월 21일에 처음 보고되었습니다._
