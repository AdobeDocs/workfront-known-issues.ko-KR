# 작업: 들여쓴 작업이 들여쓴 상태로 유지되지 않음

>[!NOTE]
>
>이 문제는 2023년 1월 12일에 수정되었습니다.

사용자가 작업 목록을 보고 작업을 들여쓰면 작업이 즉시 원래(내어쓴) 상태로 돌아갑니다.

_2022년 11월 22일에 처음 보고되었습니다._
