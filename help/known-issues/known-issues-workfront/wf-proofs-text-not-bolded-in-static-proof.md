# 증명: 정적 증명의 텍스트 형식이 웹 사이트의 형식과 일치하지 않음

<!--WF, WFP TOCs-->

>[!NOTE]
>
>이 문제는 2023년 9월 21일에 수정되었습니다.

사용자가 웹 사이트에서 정적 증명을 만들면 증명에서 웹 사이트가 굵은 글꼴로 표시되지 않습니다.

_2023년 8월 23일에 처음 보고되었습니다._
