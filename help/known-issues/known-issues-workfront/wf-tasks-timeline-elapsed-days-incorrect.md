# 작업: 하위 작업이 있는 상위 작업의 경과 일수가 올바르지 않음

상위 작업에 경과 일수 계산에 포함된 하위 작업이 있는 경우, 상위 작업의 경과 일수가 하위 작업의 경과 일수를 합친 것보다 더 적은 것으로 잘못 표시됩니다.
