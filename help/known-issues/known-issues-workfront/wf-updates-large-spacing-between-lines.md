# 업데이트: 줄 사이에 큰 공백

사용자가 Return 또는 Enter 키를 사용하여 여러 줄로 업데이트를 입력하거나 업데이트에 여러 줄을 붙여넣으면 업데이트가 예상대로 표시됩니다. 그러나 사용자가 보고서에서 해당 업데이트를 볼 경우 줄 사이에 공백이 커집니다.

이는 새로운 댓글 환경에서 보고되었습니다.
