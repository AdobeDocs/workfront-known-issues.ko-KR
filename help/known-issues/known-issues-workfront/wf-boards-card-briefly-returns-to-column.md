# 보드: 카드가 잠시 후 원래 열로 돌아감

>[!NOTE]
>
>이 문제는 2024년 4월 30일 수요일에 해결되었습니다.

사용자가 보드의 다른 열로 카드를 이동하면 잠시 후 카드가 해당 열로 이동하지 않고 원래 열로 돌아갑니다.

_2024년 3월 20일에 처음 보고되었습니다._
