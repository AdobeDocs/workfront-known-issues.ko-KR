# 작업: 사용자를 작업에 할당할 때 작업 역할이 목록에서 사라지지 않음

작업 역할이 작업에 할당되고 해당 작업이 작업 역할이 있는 사용자에게 할당되면 작업 역할이 할당 목록에서 사라지지 않습니다.
