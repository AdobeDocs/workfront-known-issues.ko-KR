# 보드: 사용자가 팀이 할당된 보드를 볼 수 없음

>[!NOTE]
>
>이 문제는 2024년 5월 15일 목요일에 해결되었습니다.

사용자가 팀이 속한 보드를 보려고 하면 해당 보드를 볼 수 없습니다. 이는 팀이 보드에 할당된 후 사용자가 팀에 추가되는 경우 발생합니다.

**해결 방법**

해당 사용자를 보드에 직접 추가합니다.

_2024년 4월 15일에 처음 보고되었습니다._
