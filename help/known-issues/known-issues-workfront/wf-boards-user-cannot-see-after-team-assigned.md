# 보드: 사용자가 팀이 할당된 보드를 볼 수 없음

사용자가 팀이 할당된 보드를 보려고 할 때 보드를 볼 수 없습니다. 이 문제는 팀이 보드에 할당된 후 사용자가 팀에 추가되는 경우에 발생합니다.

**해결 방법**

사용자를 보드에 직접 추가합니다.

_2024년 4월 15일에 처음 보고되었습니다._
