# 설정: 작업 역할 비용/시간 또는 청구/시간을 편집할 수 없음

>[!NOTE]
>
>이 문제는 미리보기 환경에서만 찾을 수 있습니다.

사용자가 작업 역할을 편집하고 비용/시간 또는 청구/시간 필드를 편집하려고 하면 필드가 비어 있고 편집할 수 없습니다.

**해결 방법**

작업 역할 목록에서 필드 인라인을 편집합니다.

_2023년 6월 29일에 처음 보고되었습니다._
