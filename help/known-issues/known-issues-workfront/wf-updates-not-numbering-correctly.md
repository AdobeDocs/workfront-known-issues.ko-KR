# 업데이트: 번호 목록의 번호가 올바르게 지정되지 않음

사용자가 번호 매기기 목록을 포함하는 댓글을 제출하면 해당 목록에 업데이트에 잘못된 번호 매기기가 표시됩니다.

이는 새로운 댓글 환경에서 보고되었습니다.

_2024년 3월 14일 금요일에 처음 보고되었습니다._
