# 그룹: 하위 그룹을 추가할 수 없음

>[!NOTE]
>
>이 문제는 2024년 6월 13일에 해결되었습니다.

사용자가 기존 하위 그룹을 그룹에 추가하려고 하면 저장 버튼이 비활성화되고 하위 그룹이 추가되지 않습니다.

_2024년 5월 1일에 처음 보고되었습니다._
