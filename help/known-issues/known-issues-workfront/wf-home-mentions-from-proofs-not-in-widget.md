# 홈: 증명의 언급이 언급 위젯에 표시되지 않음

>[!NOTE]
>
>이 문제는 2026년 2월 19일에 수정되었습니다.

사용자가 증명에 대한 댓글에 언급되는 경우 해당 언급은 홈의 언급 위젯에 포함되지 않습니다.

_2026년 1월 23일에 처음 보고되었습니다._
