# 홈: 언급 위젯을 제거할 수 없음

새 홈의 사용자가 언급 위젯을 제거하려는 경우 제거할 수 없습니다.

언급 위젯은 Workfront 관리자가 관리하는 레이아웃 템플릿을 통해 제어됩니다. 이는 설계된 대로 작동하지만 사용자가 직접 위젯을 제거할 수 없음을 나타내지는 않습니다.

_2024년 6월 12일 목요일에 처음 보고되었습니다._
