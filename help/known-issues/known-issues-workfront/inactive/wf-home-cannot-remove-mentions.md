# 홈: 언급 위젯을 제거할 수 없음

새로운 홈의 사용자가 언급 위젯을 제거하려고 하면 제거되지 않습니다.

언급 위젯은 Workfront 관리자가 관리하는 레이아웃 템플릿을 통해 제어됩니다. 이는 설계대로 작동하지만 사용자가 위젯을 직접 제거할 수 없다는 알림이 표시되지 않습니다.

_2024년 6월 12일에 처음 보고되었습니다._
