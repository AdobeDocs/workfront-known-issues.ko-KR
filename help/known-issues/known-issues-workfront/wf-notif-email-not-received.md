# 알림: 사용자가 댓글을 태그 지정할 때 이메일 알림이 수신되지 않음

>[!NOTE]
>
>이 문제는 2023년 11월 30일에 해결되었습니다.

사용자가 댓글을 태그 지정할 때 해당 댓글에 대한 이메일 알림이 수신되지 않습니다.

인앱 알림은 예상대로 작동합니다.

_2023년 24월 10일에 처음 보고되었습니다._
