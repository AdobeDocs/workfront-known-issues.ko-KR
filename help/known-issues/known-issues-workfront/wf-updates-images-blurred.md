# 업데이트: 업데이트의 이미지 미리보기가 흐릿하게 표시됨

>[!NOTE]
>
>이 문제는 2024년 4월 18일에 해결되었습니다.


사용자가 업데이트를 보고 이미지를 미리 보기 위해 이미지의 돋보기를 클릭하면 열리는 미리보기가 매우 픽셀화되어 흐릿하게 표시됩니다.

사용자가 이미지를 다운로드하면 이미지가 예상 해상도로 표시됩니다.

_2024년 3월 27일에 처음 보고되었습니다._
