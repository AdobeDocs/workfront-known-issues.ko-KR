# 업데이트: 업데이트에서 이미지 미리 보기가 흐릿합니다

사용자가 업데이트를 볼 때 이미지의 돋보기를 클릭하여 이미지를 미리 볼 때 열리는 미리 보기의 픽셀이 매우 흐리게 표시됩니다.

이미지를 다운로드하면 이미지가 예상 해상도로 나타납니다.

_2024년 3월 27일 목요일에 처음 보고되었습니다._
