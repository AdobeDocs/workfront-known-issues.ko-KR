# 대시보드: 사용자 정의 대시보드가 열리지 않음

>[!NOTE]
>
>이 문제는 2025년 4월 10일 금요일에 해결되었습니다.

사용자가 오브젝트를 볼 때 오브젝트에 첨부된 사용자 정의 대시보드를 열려고 하면 대시보드가 열리지 않습니다.

_2025년 2월 13일 금요일에 처음 보고되었습니다._
