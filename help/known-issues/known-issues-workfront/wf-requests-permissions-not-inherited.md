# 요청: 요청을 만들 때 권한이 상속되지 않음

>[!NOTE]
>
>이 문제는 2025년 12월 19일 토요일에 해결되었습니다.

사용자가 요청을 만들 때 해당 요청은 권한을 상속하지 않습니다. 즉, 사용자가 요청에 액세스하지 못할 수 있습니다.

_2025년 11월 20일 금요일에 처음 보고되었습니다._
