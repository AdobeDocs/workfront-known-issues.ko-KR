# 프로젝트: 프로젝트 헤더의 504 오류

사용자가 프로젝트를 조회할 때 프로젝트 헤더에 이름 또는 데이터가 없습니다. 대신 사용자에게 다음과 유사한 오류 메시지가 표시됩니다.

“504: 정의되지 않음 /internal/qs/customizableHeaderData?objCode=PROJ&amp;objectID=xxxxxxxxxxxxxxx”
