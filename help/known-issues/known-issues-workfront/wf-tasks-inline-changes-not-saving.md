# 작업: 작업에 대한 인라인 편집 내용이 저장되지 않음

>[!NOTE]
>
>* 이 문제는 2025년 12월 15일에 해결되었습니다.
>* 이 문제는 이전 작업 경험에 적용됩니다.

사용자가 이전 작업 경험에서 작업을 인라인으로 변경하고 저장할 때, 변경 사항이 저장되지 않습니다.

**해결 방법**

일부 변경 사항에 대해서는 새로운 작업 경험을 사용할 수 있습니다.

_2025년 12월 4일에 처음 보고되었습니다._
