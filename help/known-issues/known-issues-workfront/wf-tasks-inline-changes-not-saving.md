# 작업: 작업에 대한 인라인 편집이 저장되지 않음

>[!NOTE]
>
>이 문제는 레거시 작업 환경에 적용됩니다.

사용자가 이전 작업 경험의 작업을 인라인 변경하여 저장할 때 변경 사항이 저장되지 않습니다.

**해결 방법**

일부 변경 사항의 경우 새로운 작업 환경을 사용할 수 있습니다.

_2025년 12월 4일 금요일에 처음 보고되었습니다._
