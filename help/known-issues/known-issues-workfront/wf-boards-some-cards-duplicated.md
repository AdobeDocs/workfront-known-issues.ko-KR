# 보드: 중복 카드 관련 문제

보드에 있는 중복 카드와 관련하여 다음 문제가 보고되었습니다.

* 카드가 두 번 표시됩니다. 이 문제는 페이지를 새로 고침하여 해결할 수 있습니다.
* 사용자가 중복 카드 중 하나를 삭제하는 경우 해당 중복 카드의 모든 인스턴스가 삭제됩니다.
