# 알림: 댓글에 언급되어 있으면 사용자가 이메일 알림을 받지 않습니다

>[!NOTE]
>
>이 문제는 재현할 수 없어서 종결되었다.

@ mention 형식을 사용하는 주석에서 사용자가 언급되면, 사용자는 해당 설명을 알리는 이메일을 수신하지 않습니다. 이는 직접 업데이트 시 사용자가 이메일 알림을 포함하도록 설정을 활성화한 경우에도 발생할 수 있습니다.

_2022년 7월 27일에 처음 보고되었습니다._
