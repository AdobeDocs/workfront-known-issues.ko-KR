# 업데이트: 사용자에게 지정 업데이트에 대한 알림이 전송되지 않음

사용자가 댓글을 달고 다른 사용자를 언급하거나 태그 지정할 때 태그 지정된 사용자는 댓글에 대한 이메일 또는 인앱 알림을 받지 않습니다.

_2024년 7월 23일 수요일에 처음 보고되었습니다._
