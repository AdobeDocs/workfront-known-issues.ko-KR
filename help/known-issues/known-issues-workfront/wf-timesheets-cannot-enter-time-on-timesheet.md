# 타임시트: 타임시트에 시간을 입력할 수 없음

사용자가 타임시트에 시간을 추가하려고 하면 프로젝트 시간 또는 작업 시간 행의 시간 상자가 회색으로 표시되고 이 상자에 시간을 입력할 수 없습니다. 일반 시간 영역에서만 시간을 입력할 수 있습니다.

**해결 방법**

각 프로젝트 또는 작업에서 시간을 입력합니다.
