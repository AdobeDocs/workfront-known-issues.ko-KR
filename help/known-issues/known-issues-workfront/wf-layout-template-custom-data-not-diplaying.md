# 레이아웃 템플릿: 레이아웃 템플릿을 통해 작업 요약에 추가할 때 사용자 정의 데이터 필드가 표시되지 않음

관리자가 레이아웃 템플릿을 통해 작업 요약 섹션에 사용자 정의 데이터 필드를 추가하면 해당 필드가 작업의 요약 섹션을 보는 사용자에게 비어 있는 것으로 표시됩니다.
