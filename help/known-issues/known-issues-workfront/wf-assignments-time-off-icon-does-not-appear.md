# 할당: 작업을 할당할 때 휴무 아이콘이 표시되지 않음

사용자가 작업을 할당할 때 작업 중에 휴무가 예약된 할당자의 이름을 입력하기 시작하면 [제안된 할당] 영역 아래의 사용자 아이콘에 휴무(비행기) 아이콘이 표시되지 않습니다. 사용자 및 팀 영역 아래에 이름이 표시되도록 계속 입력하면 아이콘이 표시됩니다.

**해결 방법**

사용자 및 팀 영역에 표시될 때까지 이름을 계속 입력합니다.
