# [!UICONTROL 설정]: 선택적 상태를 숨길 수 없음

사용자가 시스템 및 그룹 수준에서 선택적 상태를 숨기려고 할 때 상태가 숨겨지지 않습니다. 사용자가 상태를 보고, 사용자가 상태를 활성화하고 변경 사항을 저장했지만 상태를 숨기는 옵션은 활성화되지 않습니다.
