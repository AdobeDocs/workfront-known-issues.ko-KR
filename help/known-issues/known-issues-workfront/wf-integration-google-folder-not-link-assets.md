# 통합:에서 폴더 연결 [!DNL Google Drive] 폴더의 자산을 연결하지 않음

사용자가에서 폴더를 연결할 때 [!DNL Google Drive] Workfront에 해당 폴더의 자산이 연결되지 않습니다. 폴더의 에셋 수는 0으로 표시됩니다.

_2024년 6월 1일에 처음 보고되었습니다._
