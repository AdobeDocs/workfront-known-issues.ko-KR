# 통합: [!DNL Google Drive]에서 폴더를 연결해도 폴더의 자산이 연결되지 않음

사용자가 [!DNL Google Drive]에서 폴더를 Workfront에 연결해도 해당 폴더의 자산이 연결되지 않습니다. 폴더의 자산 수가 0으로 표시됩니다.

_2024년 6월 1일에 처음 보고되었습니다._
