# 보드: 여러 스프린트에 카드가 표시됨

>[!NOTE]
>
>이 문제는 2024년 1월 12일 토요일에 해결되었습니다.

사용자가 보드에서 스프린트를 볼 때 다른 스프린트에 있는 카드가 보드에 표시됩니다. 이 문제는 간헐적으로 발생합니다.

_2023년 11월 1일에 처음 보고되었습니다._
