# 사용자: 부하 직원을 제거할 수 없음

사용자가 사용자로부터 직접 보고서를 제거하려고 할 때 값이 이미 있으면 직접 보고서 필드를 업데이트할 수 없습니다.

**해결 방법**

다음 중 하나를 수행하십시오.

* 새 값을 추가한 다음 값을 제거합니다.
* 사용자의 조직 차트에서 직접 보고서를 제거합니다.
