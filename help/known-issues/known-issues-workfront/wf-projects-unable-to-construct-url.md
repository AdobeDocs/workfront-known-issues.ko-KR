# 프로젝트: 을 사용하는 URL이 있는 프로젝트로 이동할 수 없음 [!UICONTROL 참조 ID]

사용자가 가 포함된 URL을 사용하여 프로젝트로 이동하려고 할 때 [!UICONTROL 참조 ID] 번호: 오류 메시지가 표시된 페이지로 리디렉션됩니다. URL을 사용하여 작업으로 이동 [!UICONTROL 참조 ID] 예상대로 작동합니다.
