# 프로젝트: [!UICONTROL 참조 ID]를 사용하는 URL로 프로젝트를 탐색할 수 없음

사용자가 [!UICONTROL 참조 ID] 번호가 포함된 URL을 사용하여 프로젝트로 이동하려고 하면 오류 메시지가 표시되는 페이지로 리디렉션됩니다. [!UICONTROL 참조 ID]가 있는 URL을 사용하여 작업으로 이동하면 예상대로 작동합니다.
