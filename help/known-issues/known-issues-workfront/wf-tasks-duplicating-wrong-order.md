# 작업: 작업을 복제하면 작업이 잘못된 순서로 표시됨

>[!NOTE]
>
>이 문제는 2024년 2월 22일 금요일에 해결되었습니다.

사용자가 작업 목록에서 하나 이상의 작업을 선택하고 이를 복제하면 작업 목록에 작업 순서 번호에 따른 잘못된 순서로 작업이 표시됩니다. 작업 번호는 올바르게 지정되지만 작업이 잘못된 순서로 표시됩니다.

**해결 방법**

페이지를 새로 고칩니다.

_2023년 11월 27일에 처음 보고되었습니다._
