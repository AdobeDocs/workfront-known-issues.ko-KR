# 증명: 새 문서 버전의 단순 증명에 이전 버전의 워크플로가 포함됨

사용자가 새 버전의 문서를 업로드한 후 새 버전에서 간단한 증명을 만들 때 새 증명에 이전 버전의 증명에 적용된 것과 동일한 워크플로가 포함됩니다. 문서에서 만든 단순 증명에는 워크플로가 포함되어서는 안 됩니다.
