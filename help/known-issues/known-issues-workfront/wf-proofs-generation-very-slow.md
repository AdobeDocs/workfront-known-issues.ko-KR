# 증명: 증명 생성이 느림

>[!NOTE]
>
>이 문제는 2025년 3월 19일 목요일에 해결되었습니다.

사용자가 Workfront에서 증명을 만들 때 증명을 생성하는 데 3~10분 정도 걸릴 수 있습니다. 증명이 생성되지만 느리게 생성됩니다.

_2024년 1월 21일 월요일에 처음 보고되었습니다._
