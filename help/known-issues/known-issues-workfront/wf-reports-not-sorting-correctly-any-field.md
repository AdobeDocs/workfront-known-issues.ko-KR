# 보고서: 모든 필드로 정렬할 때 보고서가 올바르게 정렬되지 않음


사용자가 작업 보고서를 실행하면 보고서가 로드되는 동안은 올바르게 정렬된 것처럼 보이지만 로딩이 완료되면 해당 보고서가 올바르게 정렬되지 않습니다.

이 정렬 문제는 특정 유형의 필드에 국한되지 않습니다.

**해결 방법**

대시보드에서 보고서를 표시하고 정렬합니다.

_2023년 6월 1일에 처음 보고되었습니다._
