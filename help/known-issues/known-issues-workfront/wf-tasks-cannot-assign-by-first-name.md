# 작업: 작업 할당에서 이름을 검색해도 사용자가 반환되지 않음

사용자가 작업을 할당할 때 이름별로 할당자를 검색하려고 하면 검색에서 사용자를 반환하지 않고 &quot;검색 결과가 없습니다&quot;라는 메시지가 표시됩니다.

**해결 방법**

전체 이름으로 검색하면 예상대로 사용자가 반환됩니다.
