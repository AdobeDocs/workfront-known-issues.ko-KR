# 작업: 작업 할당에서 이름으로 검색하면 사용자가 검색되지 않음

사용자가 작업을 할당하고 할당된 사람을 이름으로 검색하면 검색 결과에 아무 사용자도 나타나지 않으며 사용자에게는 “결과를 찾을 수 없음”이라는 메시지가 표시됩니다.

**해결 방법**

이름 전체로 검색하면 정상적으로 사용자를 찾을 수 있습니다.
