# 문서: 승인이 완료된 후 승인자로 추가된 사용자에게는 알림이 전송되지 않습니다

>[!NOTE]
>
>이 문제는 2025년 9월 4일 금요일에 수정되었습니다.

사용자가 문서를 승인한 후 승인자로 추가된 사용자가 추가되면 새로 추가된 승인자는 승인 요청에 대한 인앱 또는 이메일 알림을 받지 않습니다.

_2025년 7월 28일 화요일에 처음 보고되었습니다._
