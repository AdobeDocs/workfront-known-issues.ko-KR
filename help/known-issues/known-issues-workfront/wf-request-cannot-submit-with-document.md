# 요청: 요청을 작성할 때 문서를 업로드할 수 없음

>[!NOTE]
>
>이 문제는 2023년 2월 23일에 수정되었습니다.

사용자가 요청을 작성하고 문서를 업로드하려고 할 때 문서가 업로드되지 않고 다음과 같은 오류가 표시됩니다.

&quot;업로드 오류: 작은 파일을 만들거나 만든 후 파일을 첨부해 보십시오.&quot;

_2026년 2월 6일에 처음 보고되었습니다._
