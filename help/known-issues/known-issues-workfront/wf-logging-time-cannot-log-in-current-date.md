# 시간 기록: 사용자가 현재 날짜의 시간을 기록할 수 없음

>[!NOTE]
>
>이 문제는 2024년 4월 4일 금요일에 해결되었습니다.

사용자가 업데이트 영역에서 시간을 기록하려고 하면 현재 날짜가 회색으로 표시되고 사용자가 현재 날짜의 시간을 기록할 수 없습니다.

_2024년 3월 4일에 처음 보고되었습니다._
