# 사용자 정의 양식: 여러 값을 선택하면 드롭다운이 닫힘

>[!NOTE]
>
>이 문제는 2024년 7월 25일 금요일에 해결되었습니다.

사용자가 사용자 정의 양식 필드에서 여러 값을 선택하려고 하면 첫 번째 값을 선택한 후 드롭다운이 닫힙니다.

이 문제는 필드가 사용자 정의 양식의 표시 논리와 연결될 때 발생합니다.

_2024년 7월 9일에 처음 보고되었습니다._
