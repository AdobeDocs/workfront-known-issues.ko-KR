# 사용자 정의 양식: 요약 패널에 사용자 정의 데이터가 표시되지 않음

고객이 작업에 대한 요약 패널을 조회할 때 요약 패널에 표시되어야 하는 사용자 정의 데이터가 존재하지 않습니다. 작업의 레이아웃 템플릿에 사용자 정의 필드가 포함되어 있는 경우에도 이 문제가 발생할 수 있습니다.
