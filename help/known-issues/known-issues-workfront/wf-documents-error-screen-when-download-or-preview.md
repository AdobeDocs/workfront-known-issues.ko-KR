# 문서: 자산을 다운로드하거나 미리 볼 때 오류 화면


>[!NOTE]
>
>이 문제는 2024년 6월 27일에 해결되었습니다.

사용자가 Workfront에서 자산을 다운로드하거나 미리 보려고 하면 오류 화면이 대신 표시됩니다.

_2024년 2월 2일에 처음 보고되었습니다._
