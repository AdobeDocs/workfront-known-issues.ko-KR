# 승인: “모두” 필터가 개인이 제출한 승인을 제외함

>[!NOTE]
>
>이 문제는 설계대로 작동하는 것이므로 종결되었습니다.

홈의 내 승인 위젯이나 내 업데이트 페이지에서 승인을 볼 때 필터가 “모두”로 설정되어 있어도 사용자가 제출한 승인이 목록에 포함되지 않습니다.

_2024년 10월 16일에 처음 보고되었습니다._
