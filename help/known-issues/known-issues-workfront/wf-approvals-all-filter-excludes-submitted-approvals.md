# 승인: &quot;모두&quot; 필터는 개인적으로 제출된 승인을 제외함

>[!NOTE]
>
>이 문제는 2024년 6월 20일에 해결되었습니다.

홈의 내 승인 위젯 또는 내 업데이트 페이지에서 승인을 볼 때 필터가 &quot;모두&quot;로 설정되어 있어도 목록에 사용자가 제출한 승인이 포함되지 않습니다.

_2024년 10월 16일에 처음 보고되었습니다._
