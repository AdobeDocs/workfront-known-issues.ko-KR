# 통합: SSO를 사용할 때 사용자에게 잘못된 요청 오류가 발생함

>[!NOTE]
>
>이 문제는 2024년 5월 24일에 해결되었습니다.

사용자가 SSO가 활성화된 Outlook에 대한 Workfront 플러그인에 로그인하려고 하면 “잘못된 요청” 페이지가 표시되고 사용자는 로그인할 수 없습니다.

_2024년 5월 14일에 처음 보고되었습니다._
