# 통합: SSO를 사용할 때 사용자가 잘못된 요청 오류 발생

사용자가 SSO를 사용하여 Outlook용 Workfront 플러그인에 로그인하려고 하면 &quot;잘못된 요청&quot; 페이지가 표시되고 로그인할 수 없습니다.

_2024년 5월 14일 수요일에 처음 보고되었습니다._
