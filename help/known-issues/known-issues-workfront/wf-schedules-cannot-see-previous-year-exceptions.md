# 일정: 이전 연도의 예외를 볼 수 없음

>[!NOTE]
>
>이 문제는 2026년 1월 21일 목요일에 해결되었습니다.

사용자가 예외 달력을 볼 때 이전 연도를 보려고 하면 이전 연도를 볼 수 없습니다.

_2026년 1월 7일 목요일에 처음 보고되었습니다._
