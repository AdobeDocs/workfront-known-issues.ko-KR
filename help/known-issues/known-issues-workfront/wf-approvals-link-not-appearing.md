# 승인: 승인에 대한 링크를 사용할 수 없음

>[!NOTE]
>
>이 문제는 2025년 12월 19일 토요일에 해결되었습니다.

사용자가 오브젝트에 대한 승인자로 할당되면 해당 오브젝트에 대한 링크가 홈의 내 업데이트 영역 또는 내 승인 위젯에 표시되지 않습니다.

_2025년 10월 27일 화요일에 처음 보고되었습니다._
