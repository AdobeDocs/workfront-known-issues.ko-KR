# 보고서: 활용성 보고서의 예산 시간이 API를 통해 보고된 예산 시간과 일치하지 않음

사용자가 주어진 프로젝트에 대해 RPBGHR 개체를 API 호출하고, 해당 호출의 결과를 해당 프로젝트의 사용률 보고서와 비교해도 결과가 일치하지 않습니다.
