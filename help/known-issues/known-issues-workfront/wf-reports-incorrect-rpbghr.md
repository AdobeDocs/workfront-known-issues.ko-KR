# 보고서: 활용성 보고서의 예산 시간이 API를 통해 보고된 예산 시간과 일치하지 않음

>[!NOTE]
>
>이 문제는 2023년 12월 21일에 해결되었습니다.

사용자가 특정 프로젝트의 RPBGHR 오브젝트에 API 호출을 수행하고 해당 호출의 결과를 해당 프로젝트의 활용성 보고서와 비교할 때 결과가 일치하지 않습니다.

_2023년 9월 20일에 처음 보고되었습니다._
