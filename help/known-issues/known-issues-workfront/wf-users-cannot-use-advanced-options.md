# 사용자: 사용자를 만들 때 고급 옵션을 사용할 수 없음

>[!NOTE]
>
>이 문제는 2025년 10월 27일에 해결되었습니다.

Workfront 관리자가 사용자를 만들고 고급 옵션을 사용하려고 할 때, 해당 옵션을 사용할 수 없으며 다음 메시지가 표시됩니다.

```
Cannot read properties of undefined (reading 'sections')

Cannot read properties of undefined (reading 'sections')

Error fetching object details
```

_2025년 9월 29일에 처음 보고되었습니다._
