# 사용자 정의 양식: 서식 있는 텍스트 필드를 편집할 수 없음

사용자가 사용자 정의 양식에서 서식 있는 텍스트 필드를 편집하려고 할 때 변경 사항이 저장되지 않고 필드가 변경 사항을 반영하도록 업데이트되지 않습니다.

_2025년 12월 3일 목요일에 처음 보고되었습니다._
