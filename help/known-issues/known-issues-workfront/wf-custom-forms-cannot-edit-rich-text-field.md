# 사용자 정의 양식: 서식 있는 텍스트 필드를 편집할 수 없음

>[!NOTE]
>
>이 문제는 2025년 12월 22일에 해결되었습니다.

사용자가 사용자 정의 양식에서 서식 있는 텍스트 필드를 편집하려고 할 때 변경 사항이 저장되지 않으며 필드가 변경 사항을 반영하는 업데이트를 하지 않습니다.

_2025년 12월 3일에 처음 보고되었습니다._
