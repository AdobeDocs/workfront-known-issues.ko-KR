# [!UICONTROL 홈]: 작업이 다음 위치에 표시되지 않음: [!UICONTROL 내 작업] 위젯

사용자가 보기 [!UICONTROL 내 작업] 위젯 in [!UICONTROL 홈]에 할당된 작업 중 일부가 위젯에 표시되지 않습니다. 예를 들어 사용자가 프로젝트로 이동하여 프로젝트에서 작업이 할당되었는지 확인할 수 있지만 이러한 작업은 사용자의 [!UICONTROL 내 작업] 위젯.
