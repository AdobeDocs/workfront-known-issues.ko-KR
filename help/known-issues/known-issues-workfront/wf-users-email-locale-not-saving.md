# 사용자: 사용자의 이메일 로케일 설정이 저장되지 않음

>[!NOTE]
>
>이 문제는 2024년 8월 29일 금요일에 해결되었습니다.

관리자가 사용자의 이메일 로케일 설정을 변경하려고 하면 설정이 영어로 되돌아갑니다.

_2024년 7월 4일에 처음 보고되었습니다._
