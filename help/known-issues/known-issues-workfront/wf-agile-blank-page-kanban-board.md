# 애자일: [!UICONTROL 간판] 카드를 편집할 때 공백이 됩니다.

사용자가 [!UICONTROL 간판] 보드를 타다가 카드를 클릭하여 [!UICONTROL 간판] 보드를 새로 고치는 대신 편집할 때 공백이 생깁니다. 사용자가 페이지를 수동으로 새로 고치는 경우, [!UICONTROL 간판] 보드가 반환되고 올바른 편집이 표시됩니다.

**해결 방법**

페이지를 새로 고칩니다.
