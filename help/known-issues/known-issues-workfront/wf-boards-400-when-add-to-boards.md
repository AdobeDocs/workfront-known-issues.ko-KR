# 보드: 보드에 작업 추가 시 400 오류

사용자가 프로젝트를 조회하고 보드에 작업을 추가하려고 하면 작업이 추가되지 않고 사용자에게 다음 오류가 표시됩니다.

오류: “400: 정의되지 않음 /boards-service/graphql”.

**해결 방법**

보드에 연결된 카드를 생성하고 이를 작업에 연결합니다.

_2024년 2월 12일에 처음 보고되었습니다._
