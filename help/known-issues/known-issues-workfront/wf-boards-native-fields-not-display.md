# 보드: Workfront 기본 필드가 카드에 표시되지 않음

사용자가 기본(비사용자 정의) 필드를 표시하도록 구성된 카드를 조회할 때 해당 필드가 카드에 표시되지 않습니다.

**해결 방법**

카드를 클릭하여 해당 카드가 나타내는 오브젝트로 이동합니다.

_2024년 4월 15일에 처음 보고되었습니다._
