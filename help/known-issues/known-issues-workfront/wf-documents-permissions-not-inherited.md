# 문서: 상위 문제가 프로젝트로 변환될 때 문서 권한이 상속되지 않음

>[!NOTE]
>
>이 문제는 설계대로 작동하는 것이므로 종결되었습니다.

<!--This issue is on both WF and WFP TOCs-->

문서가 문제에 첨부되고 사용자가 문제를 프로젝트로 변환할 때 새 프로젝트의 문서에 상속된 권한이 없습니다.
