# 요청: 요청 목록에 &quot;제목 없음&quot;이 표시됨

>[!NOTE]
>
>이 문제는 2026년 2월 6일 토요일에 해결되었습니다.

사용자가 새 요청 경험에서 요청 목록을 볼 때 일부 요청의 제목 열에 &quot;제목 없음&quot;이 표시됩니다.

사용자가 이러한 요청 중 하나를 열면 해당 요청에 제목이 있음을 알 수 있습니다.

_2026년 1월 23일 토요일에 처음 보고되었습니다._
