# 상태: 이전에 삭제된 상태를 다시 사용할 수 없음

>[!NOTE]
>
>이 문제는 2024년 6월 13일 금요일에 해결되었습니다.

사용자가 이전에 삭제된 상태와 동일한 상태 키를 사용하여 상태를 만들려고 하면 상태가 만들어지지 않고 다음과 같은 메시지가 표시됩니다.

&quot;상태 키를 고유 값으로 확인하십시오.&quot;

_2024년 6월 3일에 처음 보고되었습니다._
