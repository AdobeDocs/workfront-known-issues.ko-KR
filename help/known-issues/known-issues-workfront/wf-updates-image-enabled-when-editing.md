# 업데이트: 댓글을 편집할 때 비활성화된 이미지 옵션을 사용할 수 있음

다음 이후 [!DNL Workfront] 관리자가 댓글에 이미지를 추가하는 옵션을 비활성화했습니다. 댓글을 작성할 때에는 이 옵션을 사용할 수 없습니다. 그러나 사용자가 기존 주석을 편집하는 경우 이미지 옵션을 사용할 수 있습니다.

_2024년 1월 29일 화요일에 처음 보고되었습니다._
