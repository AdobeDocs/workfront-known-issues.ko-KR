# Workfront Planning: 연결된 필드를 제거할 때 오류 발생

>[!NOTE]
>
>이 문제는 2025년 12월 22일 화요일에 해결되었습니다.

사용자가 Workfront Planning에서 연결된 필드를 제거하려고 하면 필드를 제거할 수 없고 &quot;필드를 삭제할 수 없습니다. 다시 시도하십시오.&quot;

_2025년 12월 10일 목요일에 처음 보고되었습니다._
