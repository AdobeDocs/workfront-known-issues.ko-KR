# Workfront 계획 수립: 연결된 필드를 제거하는 동안 오류 발생

>[!NOTE]
>
>이 문제는 2025년 12월 22일에 해결되었습니다.

사용자가 Workfront 계획 수립에서 연결된 필드를 제거하려고 할 때 해당 필드를 제거할 수 없으며 “필드를 삭제할 수 없습니다. 나중에 다시 시도하십시오.”라는 메시지가 표시됩니다.

_2025년 12월 10일에 처음 보고되었습니다._
