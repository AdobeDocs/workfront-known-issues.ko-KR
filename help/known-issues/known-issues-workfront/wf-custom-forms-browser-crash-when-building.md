# 사용자 정의 양식: 대형 양식을 작성할 때 브라우저 충돌

사용자가 양식 디자이너에서 큰 양식을 만들거나 편집할 때 브라우저가 충돌할 수 있고 양식을 만들거나 편집할 수 없습니다.

이 문제는 양식 디자이너에 영향을 줍니다. 양식은 예상대로 개체에 사용할 수 있습니다.
