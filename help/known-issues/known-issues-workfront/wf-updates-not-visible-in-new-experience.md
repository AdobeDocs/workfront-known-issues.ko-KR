# 업데이트: 새 댓글 달기 환경에서 일부 업데이트가 누락되었습니다

사용자가 새 댓글 달기 경험에서 업데이트를 볼 때 표시되어야 하는 댓글 중 일부가 표시되지 않습니다. 사용자가 기존 댓글 달기 경험으로 전환하면 모든 댓글이 표시됩니다.

_2023년 11월 28일 수요일에 처음 보고되었습니다._
