# 보드: 여러 사용자 필터링 시 모든 결과가 반환되지 않음

사용자가 보드의 할당 필터에서 여러 사용자를 필터링하면 모든 결과가 예상대로 반환되지 않습니다.

**해결 방법**

고급 필터를 사용하여 OR 문으로 결합된 각 사용자에 대한 필터를 만듭니다.
