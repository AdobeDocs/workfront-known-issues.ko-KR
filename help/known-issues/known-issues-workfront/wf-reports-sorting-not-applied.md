# 보고서: 보고서가 올바르게 정렬되지 않음

사용자가 작업 보고서를 실행하면 보고서가 로드되는 동안은 올바르게 정렬된 것처럼 보이지만 로딩이 완료되면 해당 보고서가 올바르게 정렬되지 않습니다.

이는 다음과 같은 상황이 모두 충족되는 경우 발생합니다.

* 보고서가 작업 보고서임
* 보고서가 사용자 정의 필드로 정렬됨
* 보고서에 그룹화가 적용되어 있음

_2023년 5월 4일에 처음 보고되었습니다._
