# 로깅 시간: 작업 또는 문제에 대한 시간을 편집할 수 없음

>[!NOTE]
>
>이 문제는 2024년 5월 2일 금요일에 해결되었습니다.

사용자가 작업 또는 문제에 대한 시간을 편집하려고 할 때 변경 사항이 저장되지 않습니다.

_2024년 4월 1일에 처음 보고되었습니다._
