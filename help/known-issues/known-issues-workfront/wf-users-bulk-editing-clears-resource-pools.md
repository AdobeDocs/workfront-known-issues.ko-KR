# 사용자: 일괄 편집 사용자가 리소스 풀 및 직속 부하 직원 제거

>[!NOTE]
>
>이 문제는 2025년 12월 9일에 해결되었습니다.

사용자를 일괄 편집할 때 리소스 풀 필드를 클릭하고 값을 입력하지 않으면 모든 편집된 사용자의 리소스 풀이 제거됩니다.

일괄 편집을 수행하면 편집된 사용자에서 바로 보고서가 제거될 수도 있습니다.

_2025년 11월 21일에 처음 보고되었습니다._
