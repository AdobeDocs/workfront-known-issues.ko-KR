# 보드: 반복을 변경할 때 카드가 지속됨

>[!NOTE]
>
>이 문제는 2024년 2월 8일 금요일에 해결되었습니다.

사용자가 보드에서 반복을 본 다음 반복을 변경할 때 사용자가 이전에 보았던 반복의 카드가 새 반복에 대해 표시됩니다.

_2023년 12월 14일에 처음 보고되었습니다._
