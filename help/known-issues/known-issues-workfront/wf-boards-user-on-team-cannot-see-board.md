# 보드: 사용자 팀이 보드에 추가되면 사용자가 보드를 볼 수 없음

사용자 팀이 보드에 추가되면 사용자가 보드를 보거나 사용하지 못할 수 있습니다. 이 문제는 팀의 다른 구성원이 보드를 보거나 사용할 수 있는 경우에도 발생할 수 있습니다.

**해결 방법**

사용자를 개별적으로 보드에 추가합니다.
