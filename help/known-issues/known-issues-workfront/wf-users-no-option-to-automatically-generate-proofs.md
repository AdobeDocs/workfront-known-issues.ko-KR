# 사용자: 사용자 프로필에서 증명을 자동으로 생성하는 옵션이 누락되었습니다.

>[!NOTE]
>
>이 문제는 2025년 12월 17일 목요일에 해결되었습니다.

사용자가 사용자 프로필을 볼 때 &quot;문서 업로드 시 자동으로 증명 생성&quot; 옵션을 사용할 수 없습니다. 이 문제는 옵션에 대한 모든 사전 요구 사항이 충족되었더라도 발생할 수 있습니다.

_2025년 12월 5일 토요일에 처음 보고되었습니다._
