# 워크로드 밸런서: 시간이 예기치 않게 재배포됨

>[!NOTE]
>
>이 문제는 2023년 7월 17일에 해결되었습니다.

특정 작업에 시간을 할당하면 해당 시간이 예기치 않게 다시 계산되어 재배포될 수 있습니다. 이 문제는 날짜나 페이지 사이를 이동하거나 페이지를 새로 고친 후에 발생할 수 있습니다.

_2023년 4월 25일에 처음 보고되었습니다._
