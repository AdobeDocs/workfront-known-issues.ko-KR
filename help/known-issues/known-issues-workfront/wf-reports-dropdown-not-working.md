# 보고서: 차트의 드롭다운과 관련된 문제

차트의 드롭다운과 관련하여 다음과 같은 문제가 보고되었습니다.

* 사용자가 드롭다운 열이 있는 보고서를 볼 때 차트 기능을 사용하면 차트 창의 드롭다운이 작동하지 않습니다.
* 사용자가 드롭다운을 사용하여 보고서에서 사용자를 할당하려고 하면 할당이 저장되지 않습니다.
