# 보고서: 차트의 드롭다운과 관련된 문제

>[!NOTE]
>
>이 문제는 2025년 10월 29일에 해결되었습니다.

차트의 드롭다운과 관련하여 다음 문제가 보고됨:

* 사용자가 드롭다운 열이 있는 보고서를 볼 때 차트 기능을 사용할 때 차트 창에서 드롭다운이 작동하지 않습니다.
* 사용자가 드롭다운을 사용하여 보고서에서 사용자를 할당하려고 할 때 할당이 저장되지 않습니다.

_2025년 10월 20일에 처음 보고되었습니다._
