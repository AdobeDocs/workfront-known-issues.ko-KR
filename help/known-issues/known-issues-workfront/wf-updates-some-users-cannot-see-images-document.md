# 업데이트: 일부 사용자는 댓글에 첨부된 이미지나 문서를 볼 수 없습니다

사용자가 오브젝트에 대한 업데이트를 볼 때 업데이트에 이미지 또는 문서가 첨부된 경우 이미지 또는 문서가 사용자의 업데이트에 표시되지 않습니다.

이 문제는 Workfront 관리자가 아닌 사용자에 대해 보고되었습니다.

_2025년 3월 3일 화요일에 처음 보고되었습니다._
