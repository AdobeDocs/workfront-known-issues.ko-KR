# 문제: 문제를 일괄 편집할 때 시간 필드에 잘못된 시간이 저장됨

사용자가 문제를 일괄 편집하고 날짜 필드에 대한 날짜와 시간을 선택하고 저장하면 해당 문제에서 이 필드에 저장되는 시간이 사용자가 선택한 시간과 다릅니다. 대신 사용자가 저장하면 시간이 UTC로 변환되는 것처럼 보입니다.

이는 보고서와 목록 모두에서 보고되었습니다.
