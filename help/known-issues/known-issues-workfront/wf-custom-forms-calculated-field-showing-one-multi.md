# 사용자 정의 양식: 계산된 필드에 잘못된 값이 표시됨

사용자가 다중 선택 외부 조회 필드 값을 참조하는 계산된 필드를 보면 해당 계산 필드에 선택된 값이 하나만 표시됩니다. 여러 값을 선택한 경우에도 이러한 문제가 발생할 수 있으며, 사용자가 필드를 편집할 때는 여러 값이 표시됩니다.
