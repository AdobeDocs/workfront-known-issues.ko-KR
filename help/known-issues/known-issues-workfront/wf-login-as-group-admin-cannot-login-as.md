# 로그인: 그룹 관리자가 다른 사용자로 로그인할 수 없음

>[!NOTE]
>
>이 문제는 2025년 3월 27일 금요일에 해결되었습니다.

그룹 관리자가 다른 사용자로 로그인을 시도할 때 해당 사용자로 로그인되지 않습니다. 대신 “다음으로 로그인됨” 배너에 그룹 관리자의 이름이 표시되고 해당 그룹 관리자에게는 자신의 계정이 표시됩니다.

_2025년 2월 18일에 처음 보고되었습니다._
