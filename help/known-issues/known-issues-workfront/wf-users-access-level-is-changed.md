# 사용자: 변경 기록 없이 시스템에 의해 변경된 액세스 수준

시스템에 의해 사용자의 액세스 수준이 예기치 않게 변경될 수 있습니다. 이 경우 표시되는 업데이트가 없으며 변경 사항이 감사 로그에 표시되지 않습니다.
