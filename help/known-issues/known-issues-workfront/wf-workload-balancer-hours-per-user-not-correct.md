# 업무 균형자: 사용자당 시간이 올바르게 계산되지 않음

>[!NOTE]
>
>이 문제는 2026년 2월 6일 토요일에 해결되었습니다.

사용자가 업무 균형자에서 사용자의 시간을 볼 때 해당 시간이 잘못 계산될 수 있습니다. 예를 들어, 주 보기에는 한 개의 시간 계산이 표시될 수 있지만, 4주 보기에는 해당 주의 다른 시간 수가 표시됩니다.

_2026년 1월 27일 수요일에 처음 보고되었습니다._
