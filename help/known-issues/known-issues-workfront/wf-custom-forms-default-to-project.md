# 사용자 정의 양식: 양식이 기본적으로 프로젝트 양식으로 설정됨

>[!NOTE]
>
>이 문제는 2024년 8월 29일에 해결되었습니다.

사용자가 사용자 정의 양식을 만들고 양식에 대한 오브젝트 유형을 선택하면 오브젝트 유형이 무시되고 양식은 프로젝트 사용자 정의 양식으로 생성됩니다.

_2024년 8월 12일에 처음 보고되었습니다._
