# 교정쇄: 교정쇄에 수신자를 추가할 수 없음

>[!NOTE]
>
>이 문제는 2024년 12월 13일에 해결되었습니다.

사용자가 공유 > 수신자 추가 옵션을 사용하여 교정쇄에 사용자를 추가하려고 할 때 사용자가 수신자의 이름을 클릭해도 해당 수신자가 추가되지 않습니다.

**해결 방법**

다른 방법을 사용하여 교정쇄를 공유하십시오.

_2024년 6월 1일에 처음 보고되었습니다._
