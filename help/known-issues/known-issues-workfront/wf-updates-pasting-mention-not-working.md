# 업데이트: 언급 복사 및 붙여넣기는 언급된 사용자에게 알리지 않습니다

사용자가 @ 형식으로 언급이 포함된 댓글을 복사한 다음 해당 댓글을 다른 개체의 업데이트 영역에 붙여넣으면 해당 댓글에 대한 알림이 전송되지 않습니다.

**해결 방법**

수동으로 사용자를 태그 지정한 다음 나머지 주석을 붙여넣습니다.
