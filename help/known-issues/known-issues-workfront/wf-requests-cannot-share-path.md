# 요청: 요청 대기열 경로를 공유할 수 없음

>[!NOTE]
>
>이 문제는 2024년 4월 18일 금요일에 해결되었습니다.

사용자가 요청 대기열 경로를 공유하려고 할 때 링크가 올바른 요청 대기열을 열지 않습니다. 대신 링크가 요청 페이지로 이동합니다.

_2024년 4월 18일 금요일에 처음 보고되었습니다._
