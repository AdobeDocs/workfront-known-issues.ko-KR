# 홈: 승인 대기 위젯에서 승인 버튼 누락

>[!NOTE]
>
>이 문제는 2025년 2월 13일에 해결되었습니다.

홈에서 승인 대기 위젯을 사용할 때 승인 결정을 선택할 수 있는 버튼이 표시되지 않습니다.

**해결 방법**

승인 결정을 진행하려면 직접 승인 페이지로 이동해야 합니다.

_2024년 11월 21일에 처음 보고되었습니다._
