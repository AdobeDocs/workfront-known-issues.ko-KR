# 사용자 정의 양식: 사용자 정의 자동 완성 필드에서 회사가 누락됨

>[!NOTE]
>
>이 문제는 설계대로 작동하는 것이므로 종결되었습니다.

회사 오브젝트를 참조하는 사용자 정의 자동 완성 필드에서 회사를 찾으려고 하면 사용자가 다른 회사와 같은 그룹에 속해 있음에도 불구하고 해당 사용자와 연관된 회사만 표시됩니다. 어떤 경우에는 제안되는 회사가 없는 경우도 있습니다.
