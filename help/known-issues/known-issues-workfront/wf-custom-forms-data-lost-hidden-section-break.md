# 사용자 지정 Forms: 문제를 프로젝트로 변환할 때 섹션 브레이크에 의해 숨겨진 필드의 데이터는 보존되지 않습니다

>[!NOTE]
>
> 이 문제는 2022년 8월 11일에 수정되었습니다.

사용자가 문제를 프로젝트로 변환하는 중, 문제에 표시 논리를 사용하여 숨길 수 있는 섹션 브레이크의 데이터가 있는 사용자 지정 양식이 포함되어 있으면 해당 섹션의 데이터가 새 프로젝트로 이동되지 않습니다.

_2022년 7월 20일에 처음 보고되었습니다._
