# 프로젝트: 완료된 프로젝트에 문제를 추가할 수 없음

>[!NOTE]
>
>이 문제는 2024년 10월 24일 금요일에 수정되었습니다.

사용자가 완료 상태의 프로젝트에 문제를 추가하려고 할 때 문제를 추가할 수 없습니다. 이 문제는 &quot;문제 추가 및 편집&quot; 옵션이 활성화된 경우에도 발생할 수 있습니다.

_2024년 10월 14일 화요일에 처음 보고되었습니다._
