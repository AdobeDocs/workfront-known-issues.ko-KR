# 요청: 요청 대기열이 여러 번 표시됨

>[!NOTE]
>
>* 이 문제는 2026년 1월 8일 금요일에 해결되었습니다.
>* 이 문제는 이전 보고 경험에서 보고되었습니다.

사용자가 요청을 만들기 시작하고 요청 대기열을 선택하면 동일한 요청 대기열이 드롭다운 목록에 여러 번 나타납니다.

_2025년 12월 30일 수요일에 처음 보고되었습니다._
