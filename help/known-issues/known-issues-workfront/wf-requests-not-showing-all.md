# 요청: &quot;모두&quot; 보기에 모든 요청이 표시되지 않음

>[!NOTE]
>
>* 이 문제는 새 요청 경험에 보고되었습니다.
>* 이 문제는 2026년 2월 18일 목요일에 해결되었습니다.

사용자가 새 요청 경험에서 요청 목록을 볼 때 보기가 모든 요청을 표시하도록 설정된 경우, 일부 요청만 표시됩니다.

_2025년 12월 16일 수요일에 처음 보고되었습니다._
