# 사용자: 사용자 생성 시 더 낮은 액세스 수준을 사용할 수 없음

사용자가 다른 사용자를 생성하는 경우 새로 생성된 사용자는 첫 번째 사용자의 액세스 수준만 사용할 수 있습니다. 새 사용자를 생성한 사용자의 권한보다 낮은 권한을 가진 모든 액세스 수준을 새 사용자에게도 할당할 수 있어야 합니다.

_2024년 1월 30일에 처음 보고되었습니다._
