# 작업: 작업을 생성할 때 계획된 시간이 저장되지 않음

>[!NOTE]
>
>이 문제는 2025년 12월 3일 목요일에 해결되었습니다.

사용자가 작업을 생성하고 계획된 시간을 입력한 다음 작업을 저장하면 계획된 시간이 저장되지 않습니다.

**해결 방법**

작업을 만든 다음 편집하여 계획된 시간을 추가합니다.

_2025년 10월 10일 토요일에 처음 보고되었습니다._
