# 보고서: 시간대가 다른 사용자가 다른 보고서 결과를 얻습니다

날짜 선택기를 사용하여 특정 날짜에 대한 보고서를 가져오는 다른 시간대의 사용자는 해당 보고서에 대해 다른 결과를 얻습니다.

**해결 방법**

날짜 선택기 대신 날짜 와일드카드 또는 보고서에 대한 상대적 날짜를 사용합니다.
