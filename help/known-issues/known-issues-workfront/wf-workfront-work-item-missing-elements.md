# [!DNL Workfront]: 오브젝트 페이지에 요소가 없습니다

사용자가 의 오브젝트에서 사용자 정의 섹션으로 이동할 때 [!DNL Workfront]: 로드되는 페이지에 일부 요소가 누락될 수 있습니다. 이러한 요소는 다음과 같습니다.

* 왼쪽 탐색 패널
* 사용자 정의 섹션이 속한 오브젝트의 이름
* 헤더의 필드 및 정보

**해결 방법**

다음 중 하나를 수행하십시오.

* 페이지를 다시 로드합니다
* 다음 페이지로 이동 [!DNL Workfront]
