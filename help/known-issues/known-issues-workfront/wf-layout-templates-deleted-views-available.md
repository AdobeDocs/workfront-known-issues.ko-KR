# 레이아웃 템플릿: 삭제된 보기를 계속 사용할 수 있음

관리자가 레이아웃 템플릿을 통해 목록에서 보기를 삭제하는 경우 해당 보기는 사용자가 사용할 수 있는 보기 목록에 계속 표시됩니다. 이 옵션을 선택하면 보기가 적용되지 않거나 데이터 없이 보기의 열이 적용됩니다.
