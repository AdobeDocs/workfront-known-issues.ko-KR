# 레이아웃 템플릿: 삭제된 보기가 여전히 사용 가능

>[!NOTE]
>
>이 문제는 2025년 2월 19일에 해결되었습니다.

관리자가 레이아웃 템플릿을 통해 목록에서 보기를 삭제하는 경우에도 해당 보기가 사용자의 사용 가능한 보기 목록에 계속 표시됩니다. 이를 선택하면 보기가 적용되지 않거나 데이터가 없는 보기 열이 적용됩니다.

_2025년 1월 21일에 처음 보고되었습니다._
