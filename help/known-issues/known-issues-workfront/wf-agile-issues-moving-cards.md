# 애자일: 애자일 보드에서 카드를 이동할 때 발생하는 문제

Agile 보드에서 카드를 이동할 때 다음과 같은 문제가 보고되었습니다.

* 애니메이션 드래그가 지연됨

* 새 열에서 카드를 예측할 수 없는 위치로 이동합니다
