# 작업 및 문제: 작업 또는 문제를 만들 때 실수로 종료합니다.

>[!NOTE]
>
> 이 문제는 2022년 9월 8일 작업에 대해 수정되었습니다.

사용자가 프로젝트에서 작업이나 문제를 생성하고 생성 팝업 외부를 클릭하면 경고 없이 팝업이 닫히고 이전에 입력한 모든 정보가 손실됩니다.

_2022년 3월 25일에 처음 보고되었습니다._
