# 리소스 플래너: 내보내기 버튼이 비활성화됨

>[!NOTE]
>
>이 문제는 2025년 6월 9일 화요일에 해결되었습니다.

사용자가 리소스 플래너에서 내보내려고 하면 버튼이 비활성화되고(회색으로 표시됨) 내보낼 수 없습니다.

_2025년 5월 20일 수요일에 처음 보고되었습니다._
