# 요청: GDrive의 요청에 문서를 업로드할 수 없음

>[!NOTE]
>
>* 이 문제는 새 요청 경험에 영향을 줍니다.
>* 이 문제는 2025년 12월 12일에 해결되었습니다.

사용자가 새로운 요청 경험에서 요청을 만들고 GDrive에서 문서를 업로드하려고 할 때, GDrive 창이 응답하지 않습니다. 사용자가 파일을 선택하거나 창을 닫을 수 없습니다.

**해결 방법**

키보드에서 Esc 키를 누릅니다. 그러면 창이 예상대로 작동합니다.

_2025년 5월 1일에 처음 보고되었습니다._
