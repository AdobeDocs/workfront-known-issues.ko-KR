# 업데이트: 요청자 사용자가 업데이트에 이미지를 추가할 수 없음

요청자 라이선스를 가진 사용자가 업데이트 영역을 볼 때 이미지를 추가하려고 하면 이미지 추가 아이콘이 없고 이미지 추가 옵션을 사용할 수 없습니다.

_2023년 9월 15일에 처음 보고되었습니다._
