# 작업: 추가 사용자가 작업에 할당되면 [!UICONTROL 계획된 시간]이 재할당됩니다.

>[!NOTE]
>
>이 문제는 2023년 10월 19일에 수정되었습니다.

작업의 다른 할당자에게 [!UICONTROL 계획된 시간]이 할당된 작업에 사용자가 할당되면 해당 작업의 [!UICONTROL 계획된 시간]이 작업의 모든 할당자에게 균등하게 배포됩니다.

_2023년 9월 25일에 처음 보고되었습니다._
