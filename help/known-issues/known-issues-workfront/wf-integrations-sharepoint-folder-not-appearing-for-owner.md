# 통합: [!DNL SharePoint] 폴더 소유자용으로 표시되지 않음

>[!NOTE]
>
>이 문제는 [!DNL Workfront] 오류가 발생했습니다.

사용자가 새 창에서 폴더를 보려고 하면 [!DNL SharePoint] [!UICONTROL (그래프 API)] 통합하면 폴더가 표시되지 않습니다. 사용자가 폴더 소유자이지만 이 문제가 발생합니다.

_2022년 8월 19일에 처음 보고되었습니다._
