# [!UICONTROL 홈]: 에서 문제 할당을 제거할 수 없습니다. [!UICONTROL 홈]

사용자가 [!UICONTROL 홈 작업 목록] 및 이 할당을 제거하려고 하면, 페이지가 새로 고쳐지고 할당이 제거되지 않습니다.

**해결 방법**

문제를 열고 문제 자체에서 할당을 제거합니다.
