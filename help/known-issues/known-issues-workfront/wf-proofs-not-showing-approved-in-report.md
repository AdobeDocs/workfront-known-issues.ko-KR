# 증명: 증명이 승인된 상태로 보고서에 표시되지 않음

사용자가 증명 승인 결정이 포함된 보고서를 볼 때 보고서에 [!UICONTROL 승인됨] 승인된 모든 증명에 대한 결정. 사용자가 증명 자체를 볼 때 승인되었음을 확인할 수 있습니다.
