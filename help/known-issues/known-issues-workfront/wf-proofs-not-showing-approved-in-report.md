# 증명: 보고서에 증명이 승인됨으로 표시되지 않음

사용자가 증명 승인 결정이 포함된 보고서를 볼 때 보고서에 승인된 모든 증명에 대한 [!UICONTROL 승인] 결정이 표시되지 않습니다. 사용자가 직접 증명을 볼 때 승인되었음을 확인할 수 있습니다.
