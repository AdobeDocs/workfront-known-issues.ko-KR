# 문제: 문제 상태가 드롭다운에 없음

>[!NOTE]
>
>이 문제는 2024년 3월 14일에 해결되었습니다.

사용자가 문제 헤더에서 문제 상태를 변경하려고 하면 사용 가능한 문제 상태 중 일부가 드롭다운에 표시되지 않습니다.

이 문제는 “[!UICONTROL 진행 중]” 상태로 보고되었습니다.

_2023년 11월 6일에 처음 보고되었습니다._
