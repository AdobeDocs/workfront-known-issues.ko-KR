# 사용자 정의 양식: 사용자 정의 양식 빌더의 사용자 정의 필드에서 지침을 제거할 수 없음

>[!NOTE]
>
> 이 문제는 2022년 8월 11일에 수정되었습니다.

사용자 정의 필드를 편집할 때 [!UICONTROL 지침] 영역에서 기존 텍스트를 제거하려고 시도한 후 필드를 저장하면 텍스트가 제거되지 않습니다. 사용자는 텍스트를 편집할 수 있지만 완전히 제거할 수 없습니다.

_2022년 7월 18일에 처음 보고되었습니다._
