# 할당: 작업 할당이 실패하고 상속된 권한이 제거됨

>[!NOTE]
>
>이 문제는 2024년 9월 19일에 수정되었습니다.

팀에 작업을 할당 시 할당에 실패하고 오류가 반환됩니다. 또한 상속된 권한이 제거되어 할당하려고 했던 작업에 사용자가 액세스하지 못하게 됩니다.

_2024년 9월 5일에 처음 보고되었습니다._
