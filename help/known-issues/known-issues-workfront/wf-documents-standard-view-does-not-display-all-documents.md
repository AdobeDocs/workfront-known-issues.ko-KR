# 문서: 문서 영역에 모든 문서가 표시되지 않음

사용자가 오브젝트의 문서 영역을 볼 때 문서에 첨부된 모든 문서가 표시되지 않습니다.

이 문제는 문서 영역의 표준 보기에만 적용됩니다. 목록 보기는 예상대로 작동합니다.

**해결 방법**

문서 영역의 목록 보기로 전환합니다.
