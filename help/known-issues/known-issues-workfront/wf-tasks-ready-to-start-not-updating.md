# 작업: 전임 작업이 완료됨에도 불구하고 시작할 준비 완료 필드가 업데이트되지 않음

>[!NOTE]
>
>이 문제는 2024년 12월 9일 화요일에 해결되었습니다.

모든 필수 전임 작업이 완료로 표시되었음에도 불구하고 전임 작업이 포함된 작업의 시작할 준비 완료 필드가 True로 업데이트되지 않습니다.

_2024년 10월 1일에 처음 보고되었습니다._
