# 프로젝트/문제: 프로젝트가 문제를 올바르게 해결하지 않음

>[!NOTE]
>
>이 문제는 2023년 8월 3일에 수정되었습니다.

사용자가 문제에 대해 해결 중인 오브젝트인 프로젝트의 상태를 변경하면 문제 상태가 프로젝트의 상태와 동일한 키와 일치하지 않는 상태로 변경됩니다.

**해결 방법**

해결 관계를 제거하고 상태를 독립적으로 설정합니다.

_2023년 7월 17일에 처음 보고되었습니다._
