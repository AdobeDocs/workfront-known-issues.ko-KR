# 실제 완료 일자가 해당 없음으로 표시됨

>[!NOTE]
>
>이는 문제가 아닌 것으로 판명되어 종결되었습니다.

프로젝트가 완료되면 실제 완료 일자가 프로젝트 완료 일자 대신 해당 없음으로 표시됩니다.

실제 완료 일자가 누락되면 잘못된 보고서가 발생할 수 있습니다.

_2024년 6월 3일 화요일에 처음 보고되었습니다._
