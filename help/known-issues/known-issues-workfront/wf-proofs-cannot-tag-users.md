# 교정쇄: 특정 사용자를 태그할 수 없음

>[!NOTE]
>
>이 문제는 2025년 2월 7일 토요일에 해결되었습니다.

사용자가 교정쇄에서 다른 사용자를 태그하려고 할 때 가끔 일부 사용자를 자동 완성 필드에서 사용할 수 없는 경우가 있습니다.

_2025년 1월 10일에 처음 보고되었습니다._
