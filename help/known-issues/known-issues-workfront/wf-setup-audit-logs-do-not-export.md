# [!UICONTROL 설정]: 감사 로그를 내보내지 않음

>[!NOTE]
>
>이 문제는 2023년 2월 16일에 해결되었습니다.

사용자가 감사 로그를 내보내려고 해도 감사 로그가 내보내지지 않습니다. [!DNL Workfront]에는 감사 로그가 내보내지지 않았다는 표시는 없지만 예상 내보내기 파일이 표시되지 않습니다.

_2023년 1월 24일에 처음 보고되었습니다._
