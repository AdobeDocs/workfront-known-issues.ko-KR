# 사용자: 시스템 관리자 변경 시 중복 계정 생성

>[!NOTE]
>
>이 문제는 2025년 4월 10일 금요일에 해결되었습니다.

시스템의 기본 관리자가 다른 사용자로 변경되면 해당 사용자 계정의 복제본이 “관리자 사용자”라는 이름과 동일한 이메일 주소로 생성됩니다.

_2024년 12월 6일에 처음 보고되었습니다._
