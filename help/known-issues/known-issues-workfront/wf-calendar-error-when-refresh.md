# 캘린더: 캘린더를 새로 고칠 때 오류 발생

>[!NOTE]
>
>이 문제는 2024년 4월 18일에 해결되었습니다.

사용자가 캘린더를 보고 페이지를 새로 고칠 때 “죄송합니다” 오류가 표시됩니다. 캘린더의 데이터가 예상대로 표시되지만 오류 메시지로 인해 가려질 수 있습니다.

_2024년 3월 13일에 처음 보고되었습니다._
