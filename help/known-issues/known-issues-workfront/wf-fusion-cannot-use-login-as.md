# Workfront Fusion: Workfront 다음으로 로그인 기능을 사용하여 연결을 인증할 수 없음

>[!NOTE]
>
>설계대로 작동하는 것이므로 문제가 되지 않습니다.

Workfront Fusion에서 Workfront 연결을 만들 때 사용자가 다른 사용자로 로그인하는 동안 연결을 인증할 수 없습니다.

_2024년 7월 18일에 처음 보고되었습니다._
