# 레이아웃 템플릿: 관리자가 홈 레이아웃 템플릿을 편집할 수 없음

관리자가 사용자의 홈 레이아웃 템플릿을 편집하려고 하면 인터페이스가 응답하지 않고 변경 사항을 저장할 수 없습니다.

**해결 방법**

편집하기 전에 반응하지 않는 템플릿을 자신에게 할당하십시오.

_2024년 10월 10일에 처음 보고되었습니다._
