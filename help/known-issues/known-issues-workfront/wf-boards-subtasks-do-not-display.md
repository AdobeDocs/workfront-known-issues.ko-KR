# 보드: 카드에 하위 작업이 포함되지 않음

사용자가 상위 작업에 대한 카드를 조회할 때 해당 상위 작업의 하위 작업이 카드에 표시되지 않습니다. 대신 카드에는 하위 작업이 0개 있다고 표시됩니다.

_2024년 2월 12일에 처음 보고되었습니다._
