# 프로젝트: 고정된 프로젝트의 일부 열에 정보가 표시되지 않음

사용자가 핀을 사용하여 고정된 프로젝트로 이동하면 오브젝트 목록(예: 작업 목록)에 빈 열이 표시될 수 있습니다. 예를 들어 할당이 이루어졌더라도 [!UICONTROL 할당] 열에 할당이 표시되지 않을 수 있습니다.
