# 문서: MP4 파일을 업로드할 수 없음

>[!NOTE]
>
>이 문제는 문제가 아니므로 종결되었습니다.

사용자가 MP4를 업로드하려고 할 때 해당 파일이 업로드되지 않고 “파일 업로드에 실패”했다는 오류 메시지가 표시됩니다.

_2022년 11월 16일에 처음 보고되었습니다._
