# 보드: 상태가 변경될 때 카드가 적절한 열로 이동하지 않음

연결된 카드의 연결된 오브젝트의 상태가 오브젝트에서 직접 변경되면 카드가 해당 열로 이동하지 않습니다. 카드에서 오브젝트 상태가 변경되거나 카드를 새 열로 드래그하면 카드가 예상대로 동작합니다.

_2023년 11월 28일 수요일에 처음 보고되었습니다._
