# 보드: 작업 스트림의 보드에서 사용자를 할당할 수 없음

>[!NOTE]
>
>이 문제는 2023년 9월 21일에 수정되었습니다.

사용자가 작업 스트림의 일부인 보드에서 작업에 다른 사용자를 할당하려고 할 때 사용자의 이름을 입력하기 시작하면 사용 가능한 사용자의 드롭다운 목록에 사용자가 표시되지 않습니다. 이 문제는 사용자가 활성 상태이고 보드와 작업 스트림 모두의 멤버인 경우에도 발생합니다.

또한 비활성화된 사용자가 드롭다운에 표시된다는 것을 알 수 있습니다.

_2023년 9월 7일에 처음 보고되었습니다._
