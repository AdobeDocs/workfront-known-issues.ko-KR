# 사용자 정의 양식: 사용자 정의 드롭다운 필드의 문제

다중 선택 필드로 활성화된 사용자 정의 드롭다운 필드에 다음과 같은 문제가 표시될 수 있습니다.

* 양식이 편집 모드가 아닌 경우 “+[!UICONTROL 추가]” 버튼이 표시되지 않습니다.
* 값이 없는 필드에는 “--[!UICONTROL 레이블 없음]--” 옵션이 표시됩니다.

**해결 방법**

다음 중 하나를 수행하십시오.

* 다중 선택 필드 옵션을 제거합니다.
* 기본값을 설정합니다.

_2023년 3월 30일에 처음 보고되었습니다._
