# 프로젝트: 문제에서 변환된 프로젝트는 포트폴리오 또는 프로그램에서 권한을 상속받지 않음

>[!NOTE]
>
>이 문제는 2023년 5월 4일에 수정되었습니다.

문제가 프로젝트로 변환되면 새 프로젝트는 프로젝트가 생성된 포트폴리오 또는 프로그램에서 권한을 상속받지 않습니다.

_2022년 10월 12일에 처음 보고되었습니다._
