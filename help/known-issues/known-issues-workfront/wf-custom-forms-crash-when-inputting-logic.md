# 사용자 정의 양식: 표시 로직을 입력할 때 양식 작성기에서 충돌이 발생함

>[!NOTE]
>
>이 문제는 재현할 수 없어 종결되었습니다.

사용자가 사용자 정의 양식을 작성하고자 표시 로직을 입력하면 양식 빌더에서 충돌이 발생하고 사용자에게 다음 메시지가 표시됩니다.

“오류가 발생했으며 문제를 해결하기 위해 노력 중입니다.”
