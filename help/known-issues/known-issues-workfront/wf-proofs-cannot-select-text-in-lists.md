# 증명: 목록 또는 글머리 기호에서 텍스트를 선택할 수 없음

<!--WF and WFP TOCs-->

>[!NOTE]
>
>이 문제는 2023년 8월 24일에 수정되었습니다.

사용자가 증명 뷰어에서 증명을 보고 목록이나 글머리 기호에 있는 텍스트를 선택하려고 하면 텍스트 선택 도구가 효과가 없으며 텍스트를 선택할 수 없습니다.

_2023년 8월 1일에 처음 보고되었습니다._
