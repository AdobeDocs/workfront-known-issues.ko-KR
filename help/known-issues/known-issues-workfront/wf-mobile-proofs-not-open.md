# Workfront Mobile: 증명을 열 수 없음

>[!NOTE]
>
>이 문제는 2024년 7월 15일에 릴리스된 앱 버전에서 해결되었습니다.

Workfront Mobile 앱에서 사용자가 증명을 열려고 할 때 회전하는 로드 중 표시가 나타나지만 증명이 로드되지 않습니다.

_2024년 7월 8일에 처음 보고되었습니다._
