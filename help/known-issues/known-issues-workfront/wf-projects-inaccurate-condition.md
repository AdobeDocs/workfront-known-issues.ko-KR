# 프로젝트: 프로젝트 진행 상태가 자동으로 지연으로 업데이트되지 않음

>[!NOTE]
>
>이 문제는 설계대로 작동하는 것이므로 종결되었습니다.

자동으로 프로젝트의 진행 상태가 지연으로 업데이트되어야 할 때, 예를 들어 계획된 완료 일자가 지났고 프로젝트가 완료되지 않은 경우 프로젝트 진행 상태가 지연으로 업데이트되지 않습니다. 이는 프로젝트 상태가 문제 발생으로 이동하지 않음을 의미하기도 합니다.

_2025년 3월 13일에 처음 보고되었습니다._
