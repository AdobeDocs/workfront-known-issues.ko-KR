# 작업: 일괄 편집 작업이 작업 할당을 변경함

>[!NOTE]
>
>이 문제는 2022년 11월 10일에 수정되었습니다.


사용자가 작업 세트의 필드를 일괄 편집하면 첫 번째 작업의 할당이 모든 작업에 적용됩니다. 이렇게 되면 이전 할당이 삭제됩니다.

_2022년 11월 7일에 처음 보고되었습니다._
