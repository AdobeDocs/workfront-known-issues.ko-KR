# 작업: 작업을 인라인 편집할 때 “액세스 권한 부족” 메시지

>[!NOTE]
>
>이 문제는 2025년 11월 21일에 해결되었습니다.

사용자가 목록 또는 보고서에서 작업을 인라인 편집할 때 작업을 편집할 수 없고 사용자에게 “액세스 권한이 부족하여 이 작업을 할당할 수 없습니다”라는 메시지가 표시됩니다.

이 문제는 사용자가 작업에 대한 권한을 상속했을 때 프로젝트 공유를 통해 할당하기 옵션이 비활성화되어 있는 경우 발생합니다.

**해결 방법**

작업을 열고 직접 편집합니다.

_2025년 11월 10일에 처음 보고되었습니다._
