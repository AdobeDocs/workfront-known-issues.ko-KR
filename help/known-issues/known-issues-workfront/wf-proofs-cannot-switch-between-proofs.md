# 증명: [!UICONTROL 증명 뷰어]에서 증명 사이를 전환할 수 없음

사용자가 [!UICONTROL 증명 뷰어]에서 증명을 조회하는 동안 다른 증명으로 전환하려고 할 때 증명 전환 버튼이 응답하지 않습니다. 사용자는 보고 있던 원래 증명으로 다시 돌아오거나 다른 증명으로 전환할 수 없습니다.
