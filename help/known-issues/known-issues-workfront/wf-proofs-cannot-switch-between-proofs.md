# 증명: 에서 증명 간을 전환할 수 없습니다. [!UICONTROL 증명 뷰어]

사용자가 [!UICONTROL 교정 뷰어]를 사용하고 다른 증명으로 전환하면 스위치 증명 버튼이 응답하지 않습니다. 사용자는 보고 있던 원본 증명 또는 다른 증명으로 다시 전환할 수 없습니다.

_2023년 3월 31일에 처음 보고되었습니다._
