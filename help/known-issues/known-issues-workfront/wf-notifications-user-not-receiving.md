# 알림: 사용자가 예상 알림을 받지 못함

사용자에게 알림을 전송해야 하는 이벤트가 발생할 때 해당 사용자는 알림을 받지 않습니다. 이 문제는 사용자 알림 설정에서 알림을 허용한 경우에도 발생할 수 있습니다.

이 문제는 인앱 알림과 이메일 알림 모두에 영향을 줍니다.

_2026년 2월 12일 금요일에 처음 보고되었습니다._
