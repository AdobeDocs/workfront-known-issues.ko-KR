# 타임시트: 타임시트가 생성되지 않음

>[!NOTE]
>
>이 문제는 2025년 12월 15일에 해결되었습니다.

타임시트가 예상대로 자동으로 생성되지 않습니다. 타임시트 프로필이 정확하고 다른 주의 타임시트가 올바르게 생성되더라도 이런 일이 발생할 수 있습니다.

_2025년 11월 3일에 처음 보고되었습니다._
