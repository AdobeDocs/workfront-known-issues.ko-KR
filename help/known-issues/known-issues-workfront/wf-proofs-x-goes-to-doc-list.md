# 교정쇄: 교정쇄 닫기가 문서 목록으로 리디렉션

사용자가 교정쇄를 보고 X 아이콘을 클릭하여 닫을 때 문서 세부 정보가 아닌 문서 목록으로 리디렉션됩니다.

이 문제는 다음 두 가지가 모두 참일 때 발생합니다.

* 사용자가 링크에서 교정쇄를 열었습니다. 문서 세부 정보에서 교정쇄를 열면 교정쇄를 닫을 때 문서 세부 정보로 돌아갑니다.
* 사용자가 아직 Adobe Unified Shell로 마이그레이션되지 않은 조직에 속합니다.

_2025년 5월 12일에 처음 보고되었습니다._
