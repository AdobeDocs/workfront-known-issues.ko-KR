# 사용자: 사용자가 자신의 알림 설정을 편집할 수 없음

>[!NOTE]
>
>이 문제는 2023년 2월 23일에 해결되었습니다.

[!UICONTROL 작업자] 라이선스를 보유한 사용자가 자신의 알림 설정을 편집하려고 하면 [!UICONTROL 알림] 옵션이 [!UICONTROL 편집] 창에 표시되지 않으며, 사용자가 설정을 편집할 수 없습니다.

_2023년 2월 9일에 처음 보고되었습니다._
