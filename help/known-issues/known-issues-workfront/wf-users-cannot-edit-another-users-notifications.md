# 사용자: 다른 사용자에 대한 알림 설정을 편집할 수 없음

>[!NOTE]
>
>이 문제는 2025년 9월 25일에 수정되었습니다.

사용자가 다른 사용자의 알림 설정을 편집하려고 할 때 설정을 편집할 수 없으며 다음 메시지가 표시됩니다.

“액세스 권한이 부족하여 재무 데이터를 편집할 수 없습니다.”

_2025년 9월 9일에 처음 보고되었습니다._
