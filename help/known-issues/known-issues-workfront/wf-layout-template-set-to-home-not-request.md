# 레이아웃 템플릿: 랜딩 페이지가 잘못됨

요청자 라이선스를 가진 사용자가 요청 페이지를 랜딩 페이지로 설정한 레이아웃 템플릿에 할당되면 해당 사용자는 대신 홈 페이지를 랜딩 페이지로 이동됩니다.

이 문제는 요청자 라이선스에 대해 보고되었습니다.

_2024년 3월 22일 토요일에 처음 보고되었습니다._
