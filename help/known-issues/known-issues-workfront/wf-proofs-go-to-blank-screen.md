# 증명: 증명 이름을 클릭하면 빈 화면이 표시됨

>[!NOTE]
>
>이 문제는 2024년 6월 27일 금요일에 해결되었습니다.

웹 증명 뷰어에서 사용자가 증명 이름을 클릭하여 증명을 열면 해당 증명 대신 빈 화면으로 이동됩니다.

_2024년 6월 20일에 처음 보고되었습니다._
