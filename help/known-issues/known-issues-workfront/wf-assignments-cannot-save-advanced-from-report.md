# 할당: 보고서에서 고급 할당 변경을 저장할 수 없음

사용자가 보고서에서 고급 할당을 변경하려고 할 때 변경 사항이 저장되지 않고 사용자에게 다음 메시지가 표시됩니다.

&quot;422: APIModel INTERNAL이 removeOldAssignments 필드를 지원하지 않음&quot;

**해결 방법**

프로젝트의 작업 목록에서 고급 할당을 변경할 수 있습니다.

_2026년 2월 4일에 처음 보고되었습니다._
