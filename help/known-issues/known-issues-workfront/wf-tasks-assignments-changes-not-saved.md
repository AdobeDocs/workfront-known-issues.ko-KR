# 작업: 작업의 할당 섹션을 편집해도 저장되지 않음

>[!NOTE]
>
>이 문제는 레거시 작업 환경에 적용됩니다.

사용자가 이전 작업 대화 상자에서 작업의 편집 섹션을 변경한 후 저장하면 변경 사항이 저장되지 않습니다.

**해결 방법**

일부 변경 사항의 경우 새로운 작업 환경을 사용할 수 있습니다.

_2025년 11월 24일 화요일에 처음 보고되었습니다._
