# Workfront Planning: 요청이 승인된 후 요청 상태가 &quot;진행 중&quot;으로 표시됩니다.

사용자가 요청 목록에서 완료된 요청을 볼 때 표시되는 요청 상태가 &quot;완료&quot; 대신 &quot;진행 중&quot;입니다.

사용자가 요청을 열면 요청이 완료 상태로 표시됩니다. 이 문제는 표시된 상태에만 영향을 주고 요청이 있는 상태에는 영향을 주지 않습니다.
