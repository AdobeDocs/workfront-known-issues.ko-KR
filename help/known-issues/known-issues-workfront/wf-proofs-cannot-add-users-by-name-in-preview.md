# 교정쇄: 미리 보기 환경에서 이름별로 교정쇄에 사용자를 추가할 수 없음

>[!NOTE]
>
>이 문제는 미리보기 샌드박스 환경에서 보고되었습니다.

사용자가 다른 사용자를 교정쇄에 추가하려고 할 때 이름별로 해당 사용자를 검색하려고 하면 이름이 표시되지 않고 사용자가 선택하여 다른 사용자를 교정쇄에 추가할 수 없습니다.

이 문제는 미리보기 샌드박스 환경을 새로 고친 후에도 발생할 수 있습니다.

_2025년 5월 5일에 처음 보고되었습니다._
