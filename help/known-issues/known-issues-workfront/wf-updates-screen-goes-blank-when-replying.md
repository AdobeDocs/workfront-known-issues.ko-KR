# 업데이트: 다른 회사의 사용자에게 회신하면 빈 화면이 표시됨

>[!NOTE]
>
>이 문제는 2024년 2월 29일에 해결되었습니다.

다른 회사의 사용자가 작성한 댓글에 사용자가 회신하려고 하면 빈 화면이 표시됩니다.

이는 해당 사용자가 다른 회사의 사용자를 볼 수 있는 권한이 없기 때문입니다.

_2024년 1월 8일에 처음 보고되었습니다._
