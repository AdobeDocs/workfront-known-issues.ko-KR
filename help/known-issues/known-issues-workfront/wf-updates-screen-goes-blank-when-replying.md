# 업데이트: 다른 회사의 사용자에게 회신할 때 화면이 깜박임

사용자가 다른 회사의 사용자가 댓글에 회신하려고 하면 화면이 비어 있습니다.

사용자가 다른 회사의 사용자를 볼 수 있는 권한이 없기 때문입니다.

_2024년 1월 8일 화요일에 처음 보고되었습니다._
