# 사용자: 교정쇄 프로필 변경 관련 문제

>[!NOTE]
>
>이 문제는 2025년 12월 17일에 해결되었습니다.

사용자의 교정쇄 프로필 변경과 관련된 다음 문제가 보고되었습니다.

* 사용자가 다른 교정쇄 프로필을 사용자에게 할당하고 저장할 때 변경 사항이 저장되지 않으며, 사용자에게 이전 교정쇄 프로필이 유지됩니다.
* 올바른 교정쇄 프로필이 할당되어 있더라도 사용자의 교정쇄 프로필이 잘못된 교정쇄 프로필로 표시될 수 있습니다.
* 사용자에게 다른 변경 사항이 적용될 때 해당 교정쇄 권한도 변경될 수 있습니다.

_2025년 11월 24일에 처음 보고되었습니다._
