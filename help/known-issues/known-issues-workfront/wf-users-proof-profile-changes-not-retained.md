# 사용자: 증명 프로필 변경 관련 문제

>[!NOTE]
>
>이 문제는 2025년 12월 17일 목요일에 해결되었습니다.

사용자의 증명 프로필 변경과 관련하여 다음과 같은 문제가 보고됨:

* 사용자가 다른 증명 프로필을 사용자에게 할당한 후 저장하면 변경 내용이 저장되지 않고 사용자가 이전 증명 프로필을 유지합니다.
* 올바른 증명 프로필이 지정되었더라도 사용자의 증명 프로필이 잘못된 증명 프로필로 표시될 수 있습니다.
* 사용자에게 다른 변경 사항이 적용되면 해당 증명 권한도 변경될 수 있습니다.

_2025년 11월 24일 화요일에 처음 보고되었습니다._
