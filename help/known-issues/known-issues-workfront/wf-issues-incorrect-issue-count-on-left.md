# 문제: 작업 왼쪽 탐색 메뉴의 문제 수가 잘못됨

>[!NOTE]
>
>이 문제는 2022년 10월 20일에 수정되었습니다.

사용자가 작업을 볼 때 왼쪽 탐색 메뉴의 [!UICONTROL 문제] 섹션에 표시된 숫자가 작업에 연결된 실제 문제 수를 정확하게 나타내지 않습니다.

_2022년 10월 12일에 처음 보고되었습니다._
