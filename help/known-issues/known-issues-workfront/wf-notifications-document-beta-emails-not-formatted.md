# 알림: 문서 승인 베타의 알림의 형식이 올바르지 않음

사용자가 새 문서 승인 베타에서 이메일을 수신하면 해당 이메일의 형식이 올바르게 지정되지 않습니다.

이 문제는 Windows용 Microsoft Outlook에서 보고되었습니다.

_2024년 3월 14일에 처음 보고되었습니다._
